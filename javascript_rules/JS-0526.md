# Avoid use of inline HTML templates
**ID:** `JS-0526` | **Link:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0526)

![Minor](https://img.shields.io/badge/severity-minor-yellow)![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

Prefer using HTML templates as an external file over inline templates. It is simpler, modular, and more readable.


## Bad Practice

```javascript
angular
    .module('myModule')
    .directive('helloWorld', function () {
        return {
            template: '<div>Hello World! <button>Say hello!</button></div>'
        };
    });
```

## Recommended

```javascript
angular
    .module('myModule')
    .directive('helloWorld', function () {
        return {
            templateUrl: 'template/helloWorld.html'
        };
    });
```
