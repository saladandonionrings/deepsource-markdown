# Invalid Dependent Keys
**ID:** `JS-0791` | **Lien:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0791)

![Minor](https://img.shields.io/badge/severity-minor-yellow) ![Bug Risk](https://img.shields.io/badge/type-bug_risk-green)

Invalid Dependent Keys includes following:

* Unbalanced open and closed braces.
* Unnecessary braces
* Invalid position of `@each` or `[]`
* Leading or trailing periods


## Bad Practice

```javascript
export default Component.extend({
  // Unbalanced braces:
  fullName: computed('user.{firstName,lastName', {
    // Code
  })
});
```

```javascript
export default Component.extend({
  // Unnecessary braces:
  userId: computed('user.{id}', {
    // Code
  })
});
```

```javascript
export default Component.extend({
  // Terminal `@each`:
  items: computed('arr.@each', {
    // Code
  })
});
```

```javascript
export default Component.extend({
  // `[]` in the middle:
  items: computed('arr.[].id', {
    // Code
  })
});
```

```javascript
export default Component.extend({
  // Leading period:
  userId: computed('.user.id', {
    // Code
  })
});
```

```javascript
export default Component.extend({
  // Space:
  userId: computed('user .id', {
    // Code
  })
});
```

## Recommended

```javascript
export default Component.extend({
  fullName: computed('user.{firstName,lastName}', {
    // Code
  })
});
```

```javascript
export default Component.extend({
  userId: computed('user.id', {
    // Code
  })
});
```

```javascript
export default Component.extend({
  items: computed('arr.[]', {
    // Code
  })
});
```

```javascript
export default Component.extend({
  items: computed(' [[email protected]](/cdn-cgi/l/email-protection) ', {
    // Code
  })
});
```
