# Disallow the use of reserved names in component definitions
**ID:** `JS-0707` | **Lien:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0707)

![Minor](https://img.shields.io/badge/severity-minor-yellow) ![Bug Risk](https://img.shields.io/badge/type-bug_risk-green)

There should not be name collisions between Vue components, standard HTML elements and built-in components.


## Bad Practice

```javascript
<script>
export default {
  name: 'div'
}
</script>
```

## Recommended

```javascript
<script>
export default {
  name: 'CustomNameApartFromReserverdName'
}
</script>
```
