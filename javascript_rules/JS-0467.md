# Prefer ES5 or ES6 class for returning value in render function
**ID:** `JS-0467` | **Link:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0467)

![Major](https://img.shields.io/badge/severity-major-orange)![Bug Risk](https://img.shields.io/badge/type-bug_risk-green)

`render` method of a react component must return a JSX element.

If a render method does not return valid JSX, React will throw a runtime error stating "Nothing was returned from render". To not render anything, return `null` from the render function. This trick is often used in conditional rendering.


## Bad Practice

```javascript
function Hello() {
  <div>Hello</div>;
};

class Hello extends React.Component {
  render() {
    <div>Hello</div>;
  }
}
```

## Recommended

```javascript
function Hello() {
  return <div>Hello</div>;
}

class Hello extends React.Component {
  render() {
    return <div>Hello</div>;
  }
}
```
