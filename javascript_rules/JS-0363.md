# Use `includes` method over `indexOf` method
**ID:** `JS-0363` | **Lien:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0363)

![Major](https://img.shields.io/badge/severity-major-orange) ![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

Prior to ES5, we had to use `xs.indexOf(x)` to check whether the item `x` exists in `xs` , where `xs` may be a string or an array. After ES6, JavaScript has `String#includes` and `Array#includes` .


## Bad Practice

```javascript
const str: string = "abc";
const array: number[] = [1, 2, 3];

if (str.indexOf(value) !== -1) { /* .. */ }
if (array.indexOf(value) !== -1) { /* .. */ }
```

## Recommended

```javascript
const str: string = "abc";
const array: number[] = [1, 2, 3];

if (str.includes(value)) { /* .. */ }
if (array.includes(value)) { /* .. */ }
```
