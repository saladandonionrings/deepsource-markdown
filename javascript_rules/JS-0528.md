# Found DI of specified services
**ID:** `JS-0528` | **Link:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0528)

![Minor](https://img.shields.io/badge/severity-minor-yellow)![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

Some services should be used only in a specific AngularJS service (Ajax-based service for example), in order to separate concerns. The second parameter specifies the services. The third parameter can be a list of angular objects (controller, factory, etc.). Or second parameter can be an object, where keys are angular object names and value is a list of services (like {controller: '$http', factory: '$q'})

**Rule based on Angular 1.x**


## Bad Practice

```javascript
app.controller('MyController', function($http) {
    // ...
});

app.directive('helloWorld', function($resource) {
    // ...
});
```

## Recommended

```javascript
app.controller('name', function(myService) {
    // ...
});
```
