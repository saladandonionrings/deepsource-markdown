# Bad function overloading
**ID:** `JS-0388` | **Lien:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0388)

![Minor](https://img.shields.io/badge/severity-minor-yellow) ![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

Warns for any two overloads that could be unified into one by using a union or an optional/rest parameter.


## Bad Practice

```javascript
function f(x: number): void;
function f(x: string): void;

f(): void;
f(...x: number[]): void;
```

## Recommended

```javascript
function f(x: number | string): void;

function f(x?: ...number[]): void;
```
