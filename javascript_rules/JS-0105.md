# Class methods should utilize `this`
**ID:** `JS-0105` | **Link:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0105)

![Minor](https://img.shields.io/badge/severity-minor-yellow)![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

If a class method does not use `this`, it can sometimes be made into a static function. If you do convert the method into a static function, instances of the class that call that particular method have to be converted to a static call as well `(MyClass.callStaticMethod())`


## Bad Practice

```javascript
class Person {
  sayHi() {
    const greeting = document.createElement("div");
    greeting.innerText = "Hello!";
    document.appendChild();
  }
}

const person = new Person();
person.sayHi();
```

## Recommended

```javascript
class Person {
  static sayHi() {
    const greeting = document.createElement("div");
    greeting.innerText = "Hello!";
    document.appendChild();
  }
}

Person.sayHi();
```
