# Found invalid `v-pre` directives
**ID:** `JS-0639` | **Lien:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0639)

![Major](https://img.shields.io/badge/severity-major-orange) ![Bug Risk](https://img.shields.io/badge/type-bug_risk-green)

`v-pre` is used for displaying raw mustache tags and it also helps in speeding up the compilation by skipping large numbers of nodes with no directives.

`v-pre` directive is invalid when :

* The directive has an argument.
* The directive has a modifier.
* The directive has an attribute value.

The issue is fixed by removing the argument, modifier and attribute value.


## Bad Practice

```javascript
<!-- 'v-pre' directives require no argument -->
<div v-pre:aaa/>

<!-- 'v-pre' directives require no modifier -->
<div v-pre.bbb/>

<!-- 'v-pre' directives require no attribute value -->
<div v-pre="ccc"/>
```

## Recommended

```javascript
<!-- removed modifier, attribute, argument -->
<div v-pre/>
```
