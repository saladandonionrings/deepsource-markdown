# Elements with ARIA roles must have all required attributes for that role
**ID:** `JS-0766` | **Link:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0766)

![Minor](https://img.shields.io/badge/severity-minor-yellow)![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)


## Bad Practice

```javascript
<!-- Bad: the checkbox role requires the aria-checked state -->
<span role="checkbox" aria-labelledby="foo" tabindex="0"></span>
```

## Recommended

```javascript
<!-- Good: the checkbox role requires the aria-checked state -->
<span role="checkbox" aria-checked="false" aria-labelledby="foo" tabindex="0"></span>
```
