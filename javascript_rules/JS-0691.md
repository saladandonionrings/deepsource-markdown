# Disallow unnecessary `<template>`
**ID:** `JS-0691` | **Link:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0691)

![Major](https://img.shields.io/badge/severity-major-orange)![Bug Risk](https://img.shields.io/badge/type-bug_risk-green)

In Vue.js 2.x, the `<template>` elements that have no specific directives have no effect. In Vue.js 3.x, the `<template>` elements that have no specific directives render the `<template>` elements as is, but in most cases this may not be what you intended.


## Bad Practice

```javascript
<template>...</template>
<template/>
```

## Recommended

```javascript
<template v-if="foo">...</template>
<template v-else-if="bar">...</template>
<template v-else>...</template>
<template v-for="e in list">...</template>
<template v-slot>...</template>
```
