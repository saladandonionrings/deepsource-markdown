# Prefer each component to be in its own file
**ID:** `JS-0680` | **Lien:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0680)

![Minor](https://img.shields.io/badge/severity-minor-yellow) ![Style](https://img.shields.io/badge/type-style-blue)

Whenever a build system is available to concatenate files, each component should be in its own file. This helps you to more quickly find a component when you need to edit it or review how to use it.

This should be done when the proper build system is available as recommended by the community as these build system concatenates the files.


## Bad Practice

```javascript
app.component('TodoList', {
  // ...
})

app.component('TodoItem', {
  // ...
})
```

## Recommended

```javascript
components/
|- TodoList.js
|- TodoItem.js

components/
|- TodoList.vue
|- TodoItem.vue

<script>
export default {
  name: 'my-component'
}
</script>
```
