# Require generator functions to contain yield
**ID:** `JS-0247` | **Lien:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0247)

![Minor](https://img.shields.io/badge/severity-minor-yellow) ![Bug Risk](https://img.shields.io/badge/type-bug_risk-green)

The `generator` functions should have a `yield` keyword.


## Bad Practice

```javascript
function* foo() {
  return 10;
}
```

## Recommended

```javascript
function* foo() {
  yield 5;
  return 10;
}

function foo() {
  return 10;
}

// no warnings on empty generator functions
function* foo() { }
```
