# `render` function should return value
**ID:** `JS-0622` | **Lien:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0622)

![Minor](https://img.shields.io/badge/severity-minor-yellow) ![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

It is required to have a return value for every `render` method. A render function returns a virtual DOM node, commonly named `VNode` in the Vue ecosystem, which is an interface that allows Vue to write these objects in your browser DOM. They contain all the information necessary to work with Vue.


## Bad Practice

```javascript
<script>
export default {
  render (h) {
    if (condition) {
      return h('div', 'hello')
    }
  }
}
</script>
```

## Recommended

```javascript
<script>
export default {
  render (h) {
    return h('div', 'hello')
  }
}
</script>
```
