# Detected unsupported ECMAScript built-ins on the specified version
**ID:** `JS-0267` | **Link:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0267)

![Major](https://img.shields.io/badge/severity-major-orange)![Bug Risk](https://img.shields.io/badge/type-bug_risk-green)

ECMAScript standard is updating every two months. You can check [https://node.green](https://node.green) to know which Node.js version supports each ECMAScript feature. This issue reports unsupported ECMAScript built-in variables on the configured Node.js version as lint errors.


## Bad Practice

```javascript
// ES version is 3.9.9
String.raw(s);
```

## Recommended

```javascript
// ES version is 4.0.0
String.raw(a)
```
