# Disallow unnecessary route `path` option
**ID:** `JS-0809` | **Lien:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0809)

![Minor](https://img.shields.io/badge/severity-minor-yellow) ![Bug Risk](https://img.shields.io/badge/type-bug_risk-green)

When defining a route, it's not necessary to specify the `path` option if it matches the route name.


## Bad Practice

```javascript
this.route('blog-posts', { path: '/blog-posts' });
```

## Recommended

```javascript
this.route('blog-posts');
```

```javascript
this.route('blog-posts', { path: '/blog' });
```
