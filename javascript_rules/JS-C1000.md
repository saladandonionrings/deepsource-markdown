# Avoid using multiline strings
**ID:** `JS-C1000` | **Link:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-C1000)

![Minor](https://img.shields.io/badge/severity-minor-yellow)![Style](https://img.shields.io/badge/type-style-blue)

It's possible to create multiline strings in JavaScript by using a slash before a newline. Some consider this to be a bad practice as it was an undocumented feature of JavaScript that was only formalized later. Consider using template strings or concatenating the strings instead.


## Bad Practice

```javascript
const x = "Line 1 \
         Line 2";
```

## Recommended

```javascript
const x = "Line 1\n" +
        "Line 2";

const x_ = `Line 1
Line 2`
```
