# Found empty controllers
**ID:** `JS-0524` | **Lien:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0524)

![Major](https://img.shields.io/badge/severity-major-orange) ![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

If you have an empty controller, maybe you have linked it in your Router configuration or in one of your views. You can remove this declaration because this controller has no definition, and is useless.


## Bad Practice

```javascript
angular
    .module('myModule')
    .controller('EmptyController', function () {});
```

## Recommended

```javascript
angular
    .module('myModule')
    .controller('MyController', function ($log) {
        $log.log('Hello World!');
    });
```
