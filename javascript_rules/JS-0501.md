# Found invalid file annotation
**ID:** `JS-0501` | **Link:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0501)

![Major](https://img.shields.io/badge/severity-major-orange)![Bug Risk](https://img.shields.io/badge/type-bug_risk-green)

It is recommended to use flow file annotations. Flow file annotation is simply a javascript comment with `@flow` as its content. There are possibilities of having typos or other small mistakes, which might lead to flow ignoring the file from type-checking.

The issue also raises issues for missing or misplaced annotations, common typos (e.g. `// @floww`).


## Bad Practice

```javascript
;// @flow

;
// @flow

// @Flow

// @NoFlow

// @Noflow

// @floweeeeeee

// @nofloweeeeeee
```

## Recommended

```javascript
a;

// @flow
a;

//@flow
a;

//**@flow
a;

/* data @flow bar */
a;

// @flow
a;

// @flow
// @FLow

// @noflow
a;
```
