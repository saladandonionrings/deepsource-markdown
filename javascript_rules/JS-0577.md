# Missing Banana-in-a-box template syntax
**ID:** `JS-0577` | **Link:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0577)

![Minor](https://img.shields.io/badge/severity-minor-yellow)![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

Banana in a box is a template syntax for Two-Way Data Binding. It combines the property binding and the event binding.

The name helps in remembering the order of the brackets of the Two-Way Data Binding:

* `[]`= box* `()`= banana* `[()]`= banana in a box

## Bad Practice

```javascript
<my-input [text]="val" (textChange)="val=$event"></my-input>
```

## Recommended

```javascript
<my-input [(text)]="val"></my-input>
```
