# Prefer that anchors have content and the content is accessible to screen readers
**ID:** `JS-0738` | **Lien:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0738)

![Minor](https://img.shields.io/badge/severity-minor-yellow) ![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

Making the content accessible means that it is not hidden using the `aria-hidden` prop.


## Bad Practice

```javascript
<a />
<a><TextWrapper aria-hidden /></a>
```

## Recommended

```javascript
<a>Anchor Content!</a>
<a><TextWrapper /><a>
<a dangerouslySetInnerHTML={{ __html: 'foo' }} />
```
