# Found unnecessary escape characters
**ID:** `JS-0097` | **Lien:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0097)

![Minor](https://img.shields.io/badge/severity-minor-yellow) ![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

Escaping non-special characters in strings, template literals, and regular expressions doesn't have any effect.


## Bad Practice

```javascript
"\'";
'\"';
"\#";
"\e";
`\"`;
`\"${foo}\"`;
`\#{foo}`;
/\!/;
/\@/;
```

## Recommended

```javascript
"\"";
'\'';
"\x12";
"\u00a9";
"\371";
"xs\u2111";
`\``;
`\${${foo}}`;
`$\{${foo}}`;
/\/g;
/\t/g;
/\w\$\*\^\./;
```
