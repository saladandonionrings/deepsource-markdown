# Disallow unnecessary mustache interpolations
**ID:** `JS-0716` | **Lien:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0716)

![Minor](https://img.shields.io/badge/severity-minor-yellow) ![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

The mustache interpolation with a string literal value can be changed to a static contents.


## Bad Practice

```javascript
<template>
  {{ 'Lorem ipsum' }}
  {{ "Lorem ipsum" }}
  {{ `Lorem ipsum` }}
</template>
```

## Recommended

```javascript
<template>
  Lorem ipsum
  {{ foo }}
</template>
```
