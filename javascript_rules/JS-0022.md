# Found multiple spaces in Regex
**ID:** `JS-0022` | **Link:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0022)

![Minor](https://img.shields.io/badge/severity-minor-yellow)![Bug Risk](https://img.shields.io/badge/type-bug_risk-green)

Regular expressions can sometimes be very complex and difficult to understand, which is why it's important to keep them as simple as possible to avoid mistakes.

One of the more error-prone things you can do with a regular expression is to use more than one space, leading to undesirable output as it can be due to typo error by giving extra spaces.


## Bad Practice

```javascript
let re = /deep   source/;
re = new RegExp("deep   source");
```

## Recommended

```javascript
let re = /deep {3}source/;
re = new RegExp("deep {3}source");
```
