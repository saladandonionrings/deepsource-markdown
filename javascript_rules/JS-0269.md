# Detected unsupported Node.js built-in APIs on the specified version
**ID:** `JS-0269` | **Link:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0269)

![Major](https://img.shields.io/badge/severity-major-orange)![Bug Risk](https://img.shields.io/badge/type-bug_risk-green)

Node.js is improving built-in APIs continuously. This issue reports unsupported Node.js built-in APIs on the configured Node.js version as lint errors.


## Bad Practice

```javascript
// this won't work in older versions of node.js
import assert from 'node:assert'

assert(array.length !== 0)
```

## Recommended

```javascript
// an NPM package
import Assert from 'assert-js'

Assert.notEmpty(array)
```
