# Detected the assignment to exports
**ID:** `JS-0256` | **Link:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0256)

![Major](https://img.shields.io/badge/severity-major-orange)![Bug Risk](https://img.shields.io/badge/type-bug_risk-green)

This rule is aimed at disallowing `exports = {}`, but allows `module.exports = exports = {}` to avoid conflict with node/exports-style rule's `allowBatchAssign` option.


## Bad Practice

```javascript
exports = {}
```

## Recommended

```javascript
module.exports.foo = 1
exports.bar = 2

module.exports = {}

// allows `exports = {}` if along with `module.exports =`
module.exports = exports = {}
exports = module.exports = {}
```
