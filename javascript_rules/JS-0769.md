# Prefer `tabIndex` value is not greater than zero
**ID:** `JS-0769` | **Lien:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0769)

![Minor](https://img.shields.io/badge/severity-minor-yellow) ![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

Avoid positive tabIndex property values to synchronize the flow of the page with keyboard tab order.


## Bad Practice

```javascript
<span tabIndex="5">foo</span>
<span tabIndex="3">bar</span>
<span tabIndex="1">baz</span>
<span tabIndex="2">never really sure what goes after baz</span>
```

## Recommended

```javascript
<span tabIndex="0">foo</span>
<span tabIndex="-1">bar</span>
<span tabIndex={0}>baz</span>
```
