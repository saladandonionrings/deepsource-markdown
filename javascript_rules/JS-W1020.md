# Consider using `passhref` attribute with `Link` component
**ID:** `JS-W1020` | **Lien:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-W1020)

![Major](https://img.shields.io/badge/severity-major-orange) ![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

It is recommended to pass the `href` attributes to the child components whenever the child component wraps an `<a>` tag. This can be done by using a `passHref` attribute with the `Link` component in NuxtJS. If `passhref` is not passed, then the child components won't have the `href` attributes, which would affect the SEO for the site.


## Bad Practice

```javascript
import Link from 'next/link'
import styled from 'styled-components'

// This creates a custom component that wraps an <a> tag
const RedLink = styled.a`
  color: red;
`

function NavLink({ href, name }) {
 // Must add passHref to Link otherwise href won't be pass to the children components
  return (
    <Link href={href}>
      <RedLink>{name}</RedLink>
    </Link>
  )
}

export default NavLink
```

## Recommended

```javascript
import Link from 'next/link'
import styled from 'styled-components'

// This creates a custom component that wraps an <a> tag
const RedLink = styled.a`
  color: red;
`

function NavLink({ href, name }) {

  return (
    <Link href={href} passHref>
      <RedLink>{name}</RedLink>
    </Link>
  )
}

export default NavLink
```
