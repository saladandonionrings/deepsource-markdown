# Disallow specific argument in `v-bind`
**ID:** `JS-0710` | **Lien:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0710)

![Minor](https://img.shields.io/badge/severity-minor-yellow) ![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

Spcified Arguments in `v-bind` are restricted.

E.g.


```javascript
{
  "vue/no-restricted-v-bind": ["error", "/^v-/", "foo", "bar"]
}
```

## Bad Practice

```javascript
<template>
  <div v-bind:foo="x" />
  <div :bar="x" />
</template>
```

## Recommended

```javascript
<template>
  <div v-bind:renamed_foo="x" />
  <div :renamed_bar="x" />
</template>
```
