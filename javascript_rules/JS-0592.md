# Missing component selector
**ID:** `JS-0592` | **Link:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0592)

![Minor](https://img.shields.io/badge/severity-minor-yellow)![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

Component selector must be declared. Omitting the component selector makes debugging difficult.


## Bad Practice

```javascript
@Component()
class Test {}
```

## Recommended

```javascript
@Component({
    selector: 'sg-bar-foo'
})
class Test {}
```
