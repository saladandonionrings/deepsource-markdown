# Missing `readonly` for `@Output`
**ID:** `JS-0588` | **Lien:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0588)

![Minor](https://img.shields.io/badge/severity-minor-yellow) ![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

Prefer to declare `@Output` as readonly since they are not supposed to be reassigned.


## Bad Practice

```javascript
class Test {
    @Output() testEmitter = new EventEmitter<string>();
}
```

## Recommended

```javascript
class Test {
    @Output() readonly testEmitter = new EventEmitter<string>();
}
```
