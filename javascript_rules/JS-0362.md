# Found interfaces with call signatures
**ID:** `JS-0362` | **Link:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0362)

![Minor](https://img.shields.io/badge/severity-minor-yellow)![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

Using a function type is suggested instead of an interface or object type literal with a single call signature.


## Bad Practice

```javascript
// Example 1
interface Foo {
  (): string;
}

// Example 2
function foo(bar: { (): number }): number {
  return bar();
}

// Example 3
interface Foo extends Function {
  (): void;
}
```

## Recommended

```javascript
// Example 1
interface Foo {
  (): void;
  bar: number;
}

// Example 2
function foo(bar: { (): string; baz: number }): string {
  return bar();
}

// Example 3
interface Foo {
  bar: string;
}
interface Bar extends Foo {
  (): void;
}
```
