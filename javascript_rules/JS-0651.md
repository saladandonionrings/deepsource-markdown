# Deprecation of `scope` attribute
**ID:** `JS-0651` | **Lien:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0651)

![Minor](https://img.shields.io/badge/severity-minor-yellow) ![Bug Risk](https://img.shields.io/badge/type-bug_risk-green)

Prefer `v-slot` in Vue 2.6.0+. Limited to : - `<template>` - components (for a lone default slot with props)


## Bad Practice

```javascript
<ListComponent>
    <template slot="name" scope="props">
      {{ props.title }}
    </template>
</ListComponent>
```

## Recommended

```javascript
<ListComponent>
    <template v-slot:name="props">
      {{ props.title }}
    </template>
    <template slot="name" slot-scope="props">
      {{ props.title }}
    </template>
</ListComponent>
```
