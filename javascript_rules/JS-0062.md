# Unnecessary calls to `.bind()`
**ID:** `JS-0062` | **Link:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0062)

![Minor](https://img.shields.io/badge/severity-minor-yellow)![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

The bind() method is used to create functions with specific `this` values and, optionally, binds arguments to specific values. When used to specify the value of `this`, it's important that the function actually uses `this` in its function body.


## Bad Practice

```javascript
// useless bind
const boundGetName = (function getName() {
    return "ESLint"; // correct way -> return this.name;
}).bind({ name: "ESLint" });

console.log(boundGetName());      // "ESLint"
```

```javascript
let x = function () {
    foo();
}.bind(bar);

x = (() => foo()).bind(bar);
x = (() => this.foo()).bind(bar);

x = function () {
    (function () {
      this.foo();
    }());
}.bind(bar);

x = function () {
    function foo() {
      this.bar();
    }
}.bind(baz);
```

## Recommended

```javascript
let x = (function () {
    this.foo();
}).bind({ foo() { console.log("foo") }  } );

// no uneccessary `bind`s
function f(a) {
    return a + 1;
}
```
