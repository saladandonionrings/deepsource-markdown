# Found deprecated function `tryInvoke()`
**ID:** `JS-W1001` | **Link:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-W1001)

![Minor](https://img.shields.io/badge/severity-minor-yellow)![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

Deprecate support for `tryInvoke` in Ember's Utils module (`@ember/utils`) because native JavaScript has optional chaining for developers to use as an alternative solution. Deprecating `tryInvoke` will help to reduce Ember API redundancy.

* **Function Arguments**
Arguments passed into components can be of type Function. In most cases, Function arguments should be treated as required arguments and therefore should be invoked with normal Function invocation `()`. It is important to intentionally treat Function arguments as required because in the off chance that the Function argument is `undefined`, normal Function invocation `()` will cause a runtime exception and produce a stack trace, making it easier for the developer to find the root cause of the bug.


```javascript
// app/components/parent.js
@action
parent() {
 // ...
}
```

```javascript
{{!-- app/components/parent.hbs --}}
<Child @property={{this.parent}} />
```

```javascript
// app/components/child.js
child() {
 this.args.property('value');
}
```
* **Optional Function Arguments**In the rare occasion that a Function argument is intentionally optional by design, it is recommended to use native JavaScript's optional chaining to invoke the optional Function argument`?.()`. It will avoid unintentionally treating Function arguments as optional because optional chaining invocation has the side effect of failing silently with no stack trace logged. This will cause a difficult debugging experience for the developer.

```javascript
{{!-- app/components/parent.hbs --}}
<Child @property={{this.parent}} />
```

```javascript
{{!-- app/components/some-other-parent.hbs --}}
<Child />
```

```javascript
// app/components/child.js
child() {
 this.args.property?.('value');
}
```

## Bad Practice

```javascript
import { tryInvoke } from '@ember/utils';

class DeprecatedComponent extends Component {
  containDeprecatedFunction() {
    tryInvoke(this.args, 'object', ['property']);
  }
}
```

## Recommended

```javascript
class DeprecatedComponent extends Component {
  containDeprecatedFunction() {
    this.args.object?.('property');
  }
}
```
