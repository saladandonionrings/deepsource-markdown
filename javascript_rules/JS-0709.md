# Disallow specific attribute
**ID:** `JS-0709` | **Link:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0709)

![Minor](https://img.shields.io/badge/severity-minor-yellow)![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

Specified attributes in a Vue application are restricted.

E.g.


```javascript
{
  "vue/no-restricted-static-attribute": ["error", "foo", "bar"]
}
```

## Bad Practice

```javascript
<template>
  <div foo="x" />
  <div bar />
</template>
```

## Recommended

```javascript
<template>
  <div foo_renamed="x" />
  <div bar_renamed />
</template>
```
