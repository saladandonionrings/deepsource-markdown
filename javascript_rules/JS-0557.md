# Use `$interval` instead of `setInterval`
**ID:** `JS-0557` | **Link:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0557)

![Minor](https://img.shields.io/badge/severity-minor-yellow)![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

Instead of the default setInterval function, you should use the AngularJS wrapper service $interval

**Rule based on Angular 1.x**


## Bad Practice

```javascript
// invalid
setInterval(function() {
    // ...
}, 1000) // error: You should use the $interval service instead of the default window.setInterval method

// invalid
window.setInterval(function() {
    // ...
}, 1000) // error: You should use the $interval service instead of the default window.setInterval method
```

## Recommended

```javascript
// valid
$interval(function() {
    // ...
}, 1000)
```
