# Found magic numbers
**ID:** `JS-0074` | **Link:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0074)

![Minor](https://img.shields.io/badge/severity-minor-yellow)![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

Magic numbers are numbers that occur multiple times in code without an explicit meaning. Instead, use named constants for better readability.


## Bad Practice

```javascript
let dutyFreePrice = 100,
    finalPrice = dutyFreePrice + (dutyFreePrice * 0.25);
```

## Recommended

```javascript
const TAX = 0.25;
const dutyFreePrice = 100,
    finalPrice = dutyFreePrice + (dutyFreePrice * TAX);
```
