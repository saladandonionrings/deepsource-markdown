# Found usage of deprecated name
**ID:** `JS-W1029` | **Lien:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-W1029)

![Minor](https://img.shields.io/badge/severity-minor-yellow) ![Bug Risk](https://img.shields.io/badge/type-bug_risk-green)

It is recommended not to use the deprecated names (functions, classes, variables) as using these might result in an unexpected value or error. This issue identifies deprecated function, class, method, or variable using JSDocs's comment annotation `@deprecated` or tomDoc's `Deprecated: ` .


## Bad Practice

```javascript
// <!-- calc.js -->

/**
* @deprecated use `sum` instead
*/
function add(a, b){ return a + b }

function sum(a, b){ return process(a + b); }
```

```javascript
// <!-- index.js -->

import { add } from './calc.js' // Usage of deprecated add function

console.log(add(1, 2))
```

## Recommended

```javascript
// <!-- calc.js -->

/**
* @deprecated use `sum` instead
*/
function add(a, b){ return a + b }

function sum(a, b){ return process(a + b); }
```

```javascript
// <!-- index.js -->

import { sum } from './calc.js'

console.log(sum(1, 2))
```
