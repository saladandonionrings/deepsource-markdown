# The usage of `javascript:` urls is not recommended
**ID:** `JS-0087` | **Lien:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0087)

![Minor](https://img.shields.io/badge/severity-minor-yellow) ![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

Using `javascript: URLs` is considered by some as a form of `eval` . Code passed in `javascript: URLs` has to be parsed and evaluated by the browser in the same way that `eval` is processed.


## Bad Practice

```javascript
location.href = "javascript:void(0)";
```

## Recommended

```javascript
location.href = "https://deepsource.io"
```
