# Found empty lifecycle methods
**ID:** `JS-0604` | **Lien:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0604)

![Critical](https://img.shields.io/badge/severity-critical-red) ![Bug Risk](https://img.shields.io/badge/type-bug_risk-green)

Lifecycle methods should not be empty for the following decorators.

Classes with `@Component` and `@Directive` should not have following empty methods: `ngAfterContentChecked()` , `ngAfterContentInit()` , `ngAfterViewChecked()` , `ngAfterViewInit()` , `ngDoCheck()` , `ngOnChanges()` , `ngOnDestroy()` , `ngOnInit()` .


## Bad Practice

```javascript
@Component()
class Test {
    ngOnInit() { }
}
```

## Recommended

```javascript
@Component()
class Test {
    ngAfterContentChecked() { console.log('AfterContentChecked'); }
}
```
