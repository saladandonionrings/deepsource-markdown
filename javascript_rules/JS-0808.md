# Use `render`/`clearRender` instead of `this.render`/`this.clearRender`
**ID:** `JS-0808` | **Lien:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0808)

![Minor](https://img.shields.io/badge/severity-minor-yellow) ![Bug Risk](https://img.shields.io/badge/type-bug_risk-green)

Ember's `this.render` / `this.clearRender` method and `@ember/test-helpers` 's: `render` / `clearRender` method are equivalent, but using `@ember/test-helpers` ' `render` / `clearRender` method instead of `this.render` / `this.clearRender` in tests.


## Bad Practice

```javascript
test('baz', function (assert) {
  this.render();
});
```

```javascript
test('baz', function (assert) {
  this.clearRender();
});
```

## Recommended

```javascript
import { render } from '@ember/test-helpers';

test('baz', function (assert) {
  render();
});
```

```javascript
import { clearRender } from '@ember/test-helpers';

test('baz', function (assert) {
  clearRender();
});
```
