# Default parameters should be defined at the last
**ID:** `JS-0106` | **Lien:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0106)

![Minor](https://img.shields.io/badge/severity-minor-yellow) ![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

Putting default parameter at last allows function calls to omit optional tail arguments.


```javascript
// Correct: optional argument can be omitted
function createUser(id, isAdmin = false) {}
createUser("tabby")

// Incorrect: optional argument can **not** be omitted
function createUser(isAdmin = false, id) {}
createUser(undefined, "tabby")
```

## Bad Practice

```javascript
function f(a = 0, b) {}

function f(a, b = 0, c) {}
```

## Recommended

```javascript
function f(a, b = 0) {}
```
