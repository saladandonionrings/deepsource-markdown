# Prefer not to use labels that share a name with a variable
**ID:** `JS-0121` | **Lien:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0121)

![Minor](https://img.shields.io/badge/severity-minor-yellow) ![Bug Risk](https://img.shields.io/badge/type-bug_risk-green)

Create clearer code by disallowing the bad practice of creating a label that shares a name with a variable that is in scope.


## Bad Practice

```javascript
var x = foo;
function bar() {
x:
  for (;;) {
    break x;
  }
}
```

## Recommended

```javascript
// The variable that has the same name as the label is not in scope.

function foo() {
  var q = t;
}

function bar() {
q:
  for(;;) {
    break q;
  }
}
```
