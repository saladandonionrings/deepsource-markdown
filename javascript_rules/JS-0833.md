# Syntax error
**ID:** `JS-0833` | **Lien:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0833)

![Minor](https://img.shields.io/badge/severity-minor-yellow) ![Bug Risk](https://img.shields.io/badge/type-bug_risk-green)

Found non-compliant syntax. Confirm that there are no syntax errors before committing your code to a version control system.


## Bad Practice

```javascript
// missing `=>` after arrow function parameters
const mult = (x, y) {
  return x * y
}
```

## Recommended

```javascript
const mult = (x, y) => {
  return x * y
}
```
