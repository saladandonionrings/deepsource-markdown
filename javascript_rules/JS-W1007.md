# Avoid using on commonjs in `nuxt.config.js`
**ID:** `JS-W1007` | **Link:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-W1007)

![Minor](https://img.shields.io/badge/severity-minor-yellow)![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

It is recommended not to use commonjs modules in `nuxt.config.js`. This issue prevents the use of `require/modules.exports/exports` in `nuxt.config.js`.


## Bad Practice

```javascript
<!-- nuxt.config.js -->
const { name } = require('./package.json')

module.exports = {
  mode: 'universal',
  name
}
```

## Recommended

```javascript
<!-- nuxt.config.js -->
import { name } from './package.json'

export default {
  mode: 'universal',
  name
}
```
