# Disallow variable declarations from shadowing variables declared in the outer scope
**ID:** `JS-0679` | **Lien:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0679)

![Minor](https://img.shields.io/badge/severity-minor-yellow) ![Bug Risk](https://img.shields.io/badge/type-bug_risk-green)

In Vue templates, we should avoid shadowing variables that have been declared in an outer scope.


## Bad Practice

```javascript
<div>
    <div v-for="k in 5">
      <div v-for="k in 10"></div>
      <div slot-scope="{ k }"></div>
    </div>
  </div>
```

## Recommended

```javascript
<div v-for="i in 5"></div>
<div v-for="j in 5"></div>
```
