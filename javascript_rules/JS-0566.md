# Prefer using `angular.isNumber` instead of `typeof` comparisons
**ID:** `JS-0566` | **Lien:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0566)

![Minor](https://img.shields.io/badge/severity-minor-yellow) ![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

You should use the angular.isNumber method instead of the default JavaScript implementation (typeof 3 === "object Number").

**Rule based on Angular 1.x**


## Bad Practice

```javascript
// invalid
typeof someNumber === 'number' // error: You should use the angular.isNumber method
```

## Recommended

```javascript
// valid
angular.isNumber(someNumber);
```
