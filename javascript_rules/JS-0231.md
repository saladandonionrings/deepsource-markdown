# Found duplicate class members
**ID:** `JS-0231` | **Link:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0231)

![Major](https://img.shields.io/badge/severity-major-orange)![Bug Risk](https://img.shields.io/badge/type-bug_risk-green)

If there are declarations of the same name in class members, the last declaration overwrites other declarations silently. It can cause unexpected behaviors.


```javascript
class Foo {
  bar() { console.log("hello"); }
  bar() { console.log("goodbye"); }
}

var foo = new Foo();
foo.bar(); // goodbye
```

## Bad Practice

```javascript
class Foo {
  bar() { }
  bar() { }
}

class Foo {
  bar() { }
  get bar() { }
}

class Foo {
  static bar() { }
  static bar() { }
}
```

## Recommended

```javascript
class Foo {
  bar() { }
  qux() { }
}

class Foo {
  get bar() { }
  set bar(value) { }
}

class Foo {
  static bar() { }
  bar() { }
}
```
