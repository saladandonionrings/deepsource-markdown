# Prefer event handler naming conventions in JSX
**ID:** `JS-0411` | **Lien:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0411)

![Minor](https://img.shields.io/badge/severity-minor-yellow) ![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

Ensures that any component or prop methods used to handle events are correctly prefixed.


## Bad Practice

```javascript
<MyComponent handleChange={this.handleChange} />

// Example 2
<MyComponent onChange={this.componentChanged} />
```

## Recommended

```javascript
<MyComponent onChange={this.handleChange} />

// Example 2
<MyComponent onChange={this.props.onFoo} />
```
