# Non-existent module imported
**ID:** `JS-0259` | **Link:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0259)

![Major](https://img.shields.io/badge/severity-major-orange)![Bug Risk](https://img.shields.io/badge/type-bug_risk-green)

This issue is reported in any of the following cases:

* When the module is not present in the codebase* When the module is not present in`package.json`* When the module is not a NodeJS inbuilt module.

## Bad Practice

```javascript
import typoFile from "./typo-file";   /*ERROR: "./typo-file" is not found.*/
import typoModule from "typo-module"; /*ERROR: "typo-module" is not found.*/
```

## Recommended

```javascript
import existingFile from "./existing-file";
import existingModule from "existing-module";
```
