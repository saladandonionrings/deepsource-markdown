# Detected duplicate class members
**ID:** `JS-0319` | **Lien:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0319)

![Major](https://img.shields.io/badge/severity-major-orange) ![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

If there are multiple declarations of the same name in class members, the last declaration overwrites other declarations silently. It can cause unexpected behavior.


## Bad Practice

```javascript
class Foo {
  bar() { console.log("hello"); }
  bar() { console.log("goodbye"); }
}

var foo = new Foo();
foo.bar(); // goodbye
```

## Recommended

```javascript
class Foo {
  bar() { console.log("Hello"); }
  baz() { console.log("goodbye"); }
}
```
