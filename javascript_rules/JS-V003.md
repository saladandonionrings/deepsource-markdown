# Found `this` in props default functions
**ID:** `JS-V003` | **Link:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-V003)

![Critical](https://img.shields.io/badge/severity-critical-red)![Bug Risk](https://img.shields.io/badge/type-bug_risk-green)

In Vue.js 3.0.0+, props default value factory functions no longer have access to `this`.

Instead:

* Raw props received by the component are passed to the default function as argument* The inject API can be used inside default functions

## Bad Practice

```javascript
<script>
export default {
  props: {
    a: String,
    b: {
      default () {
        /* ✗ BAD */
        return this.a
      }
    }
  }
}
</script>
```

## Recommended

```javascript
<script>
export default {
  props: {
    a: String,
    b: {
      default (props) {
        /* ✓ GOOD */
        return props.a
      }
    }
  }
}
</script>
```
