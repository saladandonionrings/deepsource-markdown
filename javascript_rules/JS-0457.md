# Definitions for unused `propTypes` detected
**ID:** `JS-0457` | **Link:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0457)

![Minor](https://img.shields.io/badge/severity-minor-yellow)![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

Avoid writing `propTypes` for properties that are not being used. It makes refactoring the codebase simpler and prevents having unused code.


## Bad Practice

```javascript
class Hello extends React.Component {
  render() {
    return <div>Hello Bob</div>;
  }
});

Hello.propTypes = {
  name: PropTypes.string
},
```

## Recommended

```javascript
class Hello extends React.Component {
  render() {
    return <div>Hello {this.props.name}</div>;
  }
};

Hello.propTypes: {
  name: PropTypes.string
},
```
