# Found multiple import of the same path
**ID:** `JS-R1000` | **Lien:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-R1000)

![Minor](https://img.shields.io/badge/severity-minor-yellow) ![Style](https://img.shields.io/badge/type-style-blue)

It is always recommended to group all imports of a single path or source in a single import declaration. These usually happen in a codebase when multiple developers are working on the same codebase. It helps improving code refactoring and code readability as well.


## Bad Practice

```javascript
// <!-- index.js -->

import calc from './calc.js'
import { add } from './calc.js'

console.log(add(1, 2))
```

## Recommended

```javascript
// <!-- index.js -->

import calc,  { add }  from './calc.js'

console.log(add(1, 2))
```
