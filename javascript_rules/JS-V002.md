# Found `key` of `<template v-for>` on child
**ID:** `JS-V002` | **Link:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-V002)

![Major](https://img.shields.io/badge/severity-major-orange)![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

Introduced as a breaking change in Vue 3 the `key` should be placed on the `<template>` tag.


## Bad Practice

```javascript
<template>
  <template v-for="todo in todos">
    <div :key="todo" />
  </template>
</template>
```

## Recommended

```javascript
<template>
  <template v-for="todo in todos" :key="todo">
    <div />
  </template>
</template>
```
