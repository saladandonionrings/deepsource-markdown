# Prefer to use `angular.isObject` instead of `typeof` comparisons
**ID:** `JS-0567` | **Lien:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0567)

![Minor](https://img.shields.io/badge/severity-minor-yellow) ![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

Prefer the `angular.isObject` method over the JavaScript `typeof` operator.

**Rule based on Angular 1.x**


## Bad Practice

```javascript
// invalid
typeof someObject === 'object' // error: You should use the angular.isObject method
```

## Recommended

```javascript
// valid
angular.isObject(someObject);
```
