# The `scope` scope should be used only on `<th>` elements
**ID:** `JS-0768` | **Lien:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0768)

![Minor](https://img.shields.io/badge/severity-minor-yellow) ![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)


## Bad Practice

```javascript
<div scope />
```

## Recommended

```javascript
<th scope="col" />
<th scope={scope} />
```
