# Found trailing undefined in function call
**ID:** `JS-W1042` | **Link:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-W1042)

![Minor](https://img.shields.io/badge/severity-minor-yellow)![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

When an argument is omitted from a function call, it will default to `undefined`. It is therefore redundant to explicitly pass an `undefined` literal as the last argument.


## Bad Practice

```javascript
function hasOptionalParam(a: number, b?: number) {
  // ...
}

hasOptionalParam(1, undefined);
```

## Recommended

```javascript
function hasOptionalParam(a: number, b?: number) {
  // ...
}

hasOptionalParam(1);
hasOptionalParam(1, 2);
```
