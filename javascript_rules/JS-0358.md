# Detected unnecessary constructors
**ID:** `JS-0358` | **Link:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0358)

![Minor](https://img.shields.io/badge/severity-minor-yellow)![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

ES2015 provides a default class constructor if one isn't explicitly specified. As such, it is unnecessary to provide an empty constructor or one that simply delegates into its parent class.


## Bad Practice

```javascript
class Animal {
    constructor () {}
}

class Dog extends Animal {
  constructor() { super() }
  bark() { /* code */ }
}
```

## Recommended

```javascript
class Animal {}; // empty abstract class
class Dog extends Animal {
  bark() { /* code */ }
};
```
