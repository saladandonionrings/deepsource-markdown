# Found empty functions
**ID:** `JS-0057` | **Link:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0057)

![Minor](https://img.shields.io/badge/severity-minor-yellow)![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

Having empty functions hurts readability, and is considered a code-smell. There's almost always a way to avoid using them. If you must use one, consider adding a comment to inform the reader of its purpose.


## Bad Practice

```javascript
getUser('SwaGaLisTiQuE', () => {})

function f() {}
```

## Recommended

```javascript
getUser('SwaGaLisTiQuE', () => {
    // empty because <reason>
})

function f() {
    // intentionally empty. <reason>
}
```
