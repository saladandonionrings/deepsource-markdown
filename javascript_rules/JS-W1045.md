# Found redundant return statement
**ID:** `JS-W1045` | **Lien:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-W1045)

![Minor](https://img.shields.io/badge/severity-minor-yellow) ![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

Return statements are unnecessary if the control flow will naturally exit a function.


## Bad Practice

```javascript
function writeToFile(file: string, content: string) {
  if (fileExists(file)) {
    fs.writeFileSync(file, content)
    return // <-- this is not necessary
  }
}
```

## Recommended

```javascript
function writeToFile(file: string, content: string) {
    if (fileExists(file)) {
        fs.writeFileSync(file, content);
    }
```
