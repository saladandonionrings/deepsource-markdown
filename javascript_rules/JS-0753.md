# Use valid `lang` value on `html` attribute
**ID:** `JS-0753` | **Link:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0753)

![Minor](https://img.shields.io/badge/severity-minor-yellow)![Bug Risk](https://img.shields.io/badge/type-bug_risk-green)

It is recommended that the `lang` prop on the `<html>` element must be a valid IETF BCP-47 language tag.


## Bad Practice

```javascript
<html>
<html lang="foo">
```

## Recommended

```javascript
<html lang="en">
<html lang="en-US">
```
