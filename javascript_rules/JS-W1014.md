# `<a>` tag used without a `Link` component
**ID:** `JS-W1014` | **Link:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-W1014)

![Major](https://img.shields.io/badge/severity-major-orange)![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

It is recommended not to use the `a` tag without wrapping it with NextJs's `Link` component. `Link` is required to enable client-side route transitions between pages and provide a single-page app experience.


## Bad Practice

```javascript
export function Home() {
  return (
    <div>
      <a href="/about">About Us</a>
    </div>
  )
}
```

## Recommended

```javascript
import Link from 'next/link'

export function Home() {
  return (
    <div>
      <Link href="/about">
        <a>About Us</a>
      </Link>
    </div>
  )
}
```
