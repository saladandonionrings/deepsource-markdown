# Forbid certain props on DOM Nodes
**ID:** `JS-0395` | **Lien:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0395)

![Major](https://img.shields.io/badge/severity-major-orange) ![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

This rule prevents passing of props to elements. This rule only applies to DOM Nodes (e.g. `<div />` ) and not Components (e.g. `<Component />` ). The list of forbidden props can be customized with the forbid option.


## Bad Practice

```javascript
// [1, { "forbid": ["id"] }]
<div id='Joe' />

// [1, { "forbid": ["style"] }]
<div style={{color: 'red'}} />
```

## Recommended

```javascript
// [1, { "forbid": ["id"] }]
<Hello id='foo' />

// [1, { "forbid": ["id"] }]
<Hello id={{color: 'red'}} />
```
