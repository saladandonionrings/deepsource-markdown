# Should not use Volatile Computed Properties
**ID:** `JS-0810` | **Link:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0810)

![Minor](https://img.shields.io/badge/severity-minor-yellow)![Bug Risk](https://img.shields.io/badge/type-bug_risk-green)

Volatile computed properties are deprecated as of Ember 3.9.


## Bad Practice

```javascript
const Person = EmberObject.extend({
  fullName: computed(function () {
    return `${this.firstName} ${this.lastName}`;
  }).volatile()
});
```

## Recommended

```javascript
const Person = EmberObject.extend({
  // Native getter:
  get fullName() {
    return `${this.firstName} ${this.lastName}`;
  }
});
```
