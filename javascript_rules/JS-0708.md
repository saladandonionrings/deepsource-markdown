# Disallow specific component option
**ID:** `JS-0708` | **Link:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0708)

![Minor](https://img.shields.io/badge/severity-minor-yellow)![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

Use of component option name defined in the list of strings or follows a pattern that is restricted.

Example Config:


```javascript
{
  "vue/no-restricted-component-options": ["error", "init", "beforeCompile", "compiled", "activate", "ready", "/^(?:at|de)tached$/"]
}
```

## Bad Practice

```javascript
<script>
export default {
  init: function () {},
  beforeCompile: function () {},
  compiled: function () {},
  activate: function () {},
  ready: function () {},
  attached: function () {},
  detached: function () {},
}
</script>
```

## Recommended

```javascript
<script>
export default {
  beforeCreate: function () {},
  activated: function () {},
  mounted: function () {},
}
</script>
```
