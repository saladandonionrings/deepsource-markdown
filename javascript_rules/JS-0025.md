# Found unreachable code
**ID:** `JS-0025` | **Link:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0025)

![Major](https://img.shields.io/badge/severity-major-orange)![Performance](https://img.shields.io/badge/type-performance-white)

Some code paths are unreachable because the `return`, `throw`, `break`, and `continue` statements unconditionally exit a block of code. The code statements after the above keywords (which exit the code block) will not execute.


## Bad Practice

```javascript
function func() {
    return true;
    next_func();
}

function add(a, b) {
    throw new Error("Oops!");
    return a + b;
}

while(value) {
    break;
    do_action();
}

throw new Error("Oops!");
console.log("done");

function check() {
    if (Math.random() < 0.5) {
        return;
    } else {
        throw new Error();
    }
    console.log("done");
}

for (;;) {}
console.log("done");
```

## Recommended

```javascript
function func() {
    return calc();
    function calc() {
        return 1;
    }
}

switch (func) {
    case 1:
        var x = get_x();
        use_x(x);
        break;
}
```
