# Prefer using the `$log` service instead of the console methods
**ID:** `JS-0559` | **Lien:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0559)

![Minor](https://img.shields.io/badge/severity-minor-yellow) ![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

You should use $log service instead of console for the methods 'log', 'debug', 'error', 'info', 'warn'

**Rule based on Angular 1.x**


## Bad Practice

```javascript
// invalid
console.log('Hello world!'); // error: You should use the "log" method of the AngularJS Service $log instead of the console object

// invalid
console.error('Some error!'); // error: You should use the "error" method of the AngularJS Service $log instead of the console object
```

## Recommended

```javascript
// valid
$log.log('Hello world!');

// valid
$log.error('Some error!');
```
