# Deprecation of `slot-scope` attribute
**ID:** `JS-0653` | **Lien:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0653)

![Major](https://img.shields.io/badge/severity-major-orange) ![Bug Risk](https://img.shields.io/badge/type-bug_risk-green)

* Prefer `v-slot` in 2.6.0+
* Limited to : `<template>` components (for a lone default slot with props)
* `<template>`
* components (for a lone default slot with props)

* `<template>`
* components (for a lone default slot with props)


## Bad Practice

```javascript
<ListComponent>
    <template slot-scope="name">
      {{ props.title }}
    </template>
</ListComponent>
```

## Recommended

```javascript
<ListComponent>
    <template v-slot:name>
      {{ props.title }}
    </template>
</ListComponent>
```
