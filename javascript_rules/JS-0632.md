# Found invalid `v-else` directives
**ID:** `JS-0632` | **Lien:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0632)

![Minor](https://img.shields.io/badge/severity-minor-yellow) ![Bug Risk](https://img.shields.io/badge/type-bug_risk-green)

`v-else` directive is not valid when:

* The directive has an argument.
* The directive has a modifier.
* The directive has an attribute value.
* The directive is on the elements that the previous element don't have `v-if` / `v-else-if` directives.
* The directive is on the elements which have `v-if` / `v-else-if` directives.


## Bad Practice

```javascript
<!-- require no argument -->
<div v-if="condition_one"></div>
<div v-else:aaa></div>

<!-- require no modifier -->
<div v-if="condition_one"></div>
<div v-else.bbb/>

<!-- require no attribute value -->
<div v-if="condition_one"></div>
<div v-else="condition_two"></div>

<!-- No previous element with v-if or v-else-if -->
<div v-else></div>

<!-- 'v-else' and 'v-if' directives can't exist on the same element -->
<div v-if="condition_one" v-else></div>
```

## Recommended

```javascript
<!-- require no argument -->
<div v-if="condition_one"></div>
<div v-else></div>

<!-- require no modifier -->
<div v-if="condition_one"></div>
<div v-else/>

<!-- remove attribute from "v-else" -->
<div v-if="condition_one"></div>
<div v-else></div>

<!-- Add previous element with v-if or v-else-if -->
<div v-if="condition_one"/>
<div v-else></div>

<!-- 'v-else' and 'v-if' directives can't exist on the same element -->
<div v-if="condition_one"</div>
<div v-else/>
```
