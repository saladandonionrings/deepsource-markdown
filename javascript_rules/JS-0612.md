# Parsing errors detected in `<template>`
**ID:** `JS-0612` | **Lien:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0612)

![Major](https://img.shields.io/badge/severity-major-orange) ![Bug Risk](https://img.shields.io/badge/type-bug_risk-green)

Syntax errors, like the following, are detected inside directives/mustaches in `<template>` block.

* Syntax errors of scripts in directives.
* Syntax errors of scripts in mustaches.
* Syntax errors of HTML.
* Invalid end tags.
* Attributes in end tags.


## Bad Practice

```javascript
<template>
  // Parsing error: Unexpected token bar
  {{ foo bar }}

  // Parsing error: Expected to be an expression, but got empty
  <div v-show=" ">hello</div>

  // Parsing error: x-invalid-end-tag.
  </div>
</template>
```

## Recommended

```javascript
<template>
  {{ `${foo} ${bar}` }}

  <div v-show="false">hello</div>

  <div></div>
</template>
```
