# `label` tags should have a text label and an associated control
**ID:** `JS-0752` | **Lien:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0752)

![Minor](https://img.shields.io/badge/severity-minor-yellow) ![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

It is recommended to add text attributes for `label` tags as it helps the browser in easing accessibility operations.

There are two supported ways to associate a label with a control:

* Wrapping a control in a label tag.
* Adding `htmlFor` to a label and assigning it a DOM ID string that indicates an input on the page.


## Bad Practice

```javascript
function Func(props) {
  return <label {...props} />
}

<input type="text" />
<label>Surname</label>
```

## Recommended

```javascript
function Func(props) {
    const {
        htmlFor,
        ...otherProps
    } = props;

   return <label htmlFor={htmlFor} {...otherProps} />
}

<label>
  <input type="text" />
  Surname
</label>
```
