# Use `$document` instead of `document`
**ID:** `JS-0555` | **Lien:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0555)

![Minor](https://img.shields.io/badge/severity-minor-yellow) ![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

Instead of the default document object, you should prefer the AngularJS wrapper service $document.

**Rule based on Angular 1.x**


## Bad Practice

```javascript
// invalid
document.title // error: You should use the $document service instead of the default document object
```

## Recommended

```javascript
// valid
$document[0].title = ""
```
