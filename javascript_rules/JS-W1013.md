# Use `head` property in component as a function
**ID:** `JS-W1013` | **Lien:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-W1013)

![Major](https://img.shields.io/badge/severity-major-orange) ![Bug Risk](https://img.shields.io/badge/type-bug_risk-green)

It is recommended to use `head` as a function as it provides access to more data through `this` .


## Bad Practice

```javascript
export default {
  head: {
    title: "My page"
  }
}
```

## Recommended

```javascript
export default {
  head() {
    return {
      title: this.title
    }
  }
}
```
