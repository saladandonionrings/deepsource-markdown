# Invalid definition of `new` and `constructor`
**ID:** `JS-0335` | **Lien:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0335)

![Minor](https://img.shields.io/badge/severity-minor-yellow) ![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

Apparent attempts to define constructors for interfaces or new for classes are warned.


## Bad Practice

```javascript
class C {
  new(): C;
}

interface I {
  new (): I;
  constructor(): void;
}
```

## Recommended

```javascript
class C {
  constructor() {}
}
interface I {
  new (): C;
}
```
