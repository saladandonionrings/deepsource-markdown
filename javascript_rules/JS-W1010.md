# Consider using `preconnect` with Google Fonts
**ID:** `JS-W1010` | **Link:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-W1010)

![Minor](https://img.shields.io/badge/severity-minor-yellow)![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

It is recommended to use `preconnect` with Google Fonts to initiate an early connection to the origin. Adding this informs the browser that your page intends to establish a connection to another origin, and that you'd like the process to start as soon as possible.


## Bad Practice

```javascript
export const Test = () => (
  <div>
    <link href="https://fonts.gstatic.com" />
  </div>
);
```

## Recommended

```javascript
export const Test = () => (
  <div>
    <link rel="preconnect" href="https://fonts.gstatic.com" />
  </div>
);
```
