# Prefer boolean attributes notation in JSX
**ID:** `JS-0400` | **Link:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0400)

![Minor](https://img.shields.io/badge/severity-minor-yellow)![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

When using a boolean attribute in JSX, you can set the attribute value to true or omit the value. This helps to keep consistency in code.


## Bad Practice

```javascript
var Hello = <Hello personal={true} />;
```

## Recommended

```javascript
var Hello = <Hello personal />;
```
