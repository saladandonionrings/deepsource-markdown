# Avoid using `<title>` with `Head` from `next/document`
**ID:** `JS-W1018` | **Link:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-W1018)

![Minor](https://img.shields.io/badge/severity-minor-yellow)![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

It is recommended not to define `<title>` within that `Head` imported from `next/document` as it should only be used for any `<head>` code that is common for all pages. Title tags should be defined at page level using `next/head`.


## Bad Practice

```javascript
import { Head } from "next/document";

export class Home {
  render() {
    return (
      <Head>
        <title>Welcome to Deepsource !</title>
      </Head>
    );
  }
}
```

## Recommended

```javascript
import Head from 'next/head'

export class Home {
  render() {
    return (
        <Head>
          <title>Welcome to Deepsource !</title>
        </Head>
    )
  }
}
```
