# Prefer to use `angular.isString` instead of `typeof` comparisons
**ID:** `JS-0568` | **Link:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0568)

![Major](https://img.shields.io/badge/severity-major-orange)![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

Prefer the `angular.isString` over the JavaScript `typeof` operator.

**Rule based on Angular 1.x**


## Bad Practice

```javascript
typeof someString === 'string'
```

## Recommended

```javascript
angular.isString(someString);
```
