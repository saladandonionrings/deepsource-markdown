# Found invalid `v-bind` directive
**ID:** `JS-0628` | **Lien:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0628)

![Minor](https://img.shields.io/badge/severity-minor-yellow) ![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

The `v-bind` directive is invalid when:

* The directive does not have an attribute value.
* The directive has invalid modifiers.


## Bad Practice

```javascript
<template>
  <div v-bind/>
  <div :aaa/>
  <div v-bind:aaa.bbb="todo"/>
</template>
```

## Recommended

```javascript
<template>
  <div v-bind="todo"/>
  <div v-bind:aaa="todo"/>
  <div :aaa="todo"/>
  <div :aaa.prop="todo"/>
</template>
```
