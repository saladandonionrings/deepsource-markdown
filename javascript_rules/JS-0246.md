# Require template literals instead of string concatenation
**ID:** `JS-0246` | **Link:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0246)

![Minor](https://img.shields.io/badge/severity-minor-yellow)![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

In ES2015 (ES6), we can use template literals instead of string concatenation.


```javascript
var str = `Hello, ${name}!`;
```

## Bad Practice

```javascript
var str = "Hello, " + name + "!";
var str = "Time: " + (12 * 60 * 60 * 1000);
```

## Recommended

```javascript
var str = "Hello World!";
var str = `Hello, ${name}!`;
var str = `Time: ${12 * 60 * 60 * 1000}`;

var str = "Hello, " + "World!";
```
