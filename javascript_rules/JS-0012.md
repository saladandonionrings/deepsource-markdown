# Found unnecessary boolean casts
**ID:** `JS-0012` | **Link:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0012)

![Minor](https://img.shields.io/badge/severity-minor-yellow)![Bug Risk](https://img.shields.io/badge/type-bug_risk-green)

In contexts such as an `if` statement's test where the result of the expression will already be coerced to a `Boolean`, casting to a `Boolean` via double negation (`!!`) or a `Boolean` call is unnecessary.


## Bad Practice

```javascript
const cond = !!!isPresent;
const cond2 = Boolean(!!isPresent);

if (!!cond) {
    // ...
}

if (Boolean(cond2)) {
    // ...
}

while (!!cond3) {
    // ...
}

do {
    // ...
} while (Boolean(cond));

for (; !!cond; ) {
    // ...
}
```

## Recommended

```javascript
const cond = !!isPresent;
const cond2 = Boolean(isPresent);

function checkCond() {
    return !!isPresent;
}

const cond3 = !!(isPresent ? flag : flag2);
```
