# Prefer to use `angular.isFunction` instead of `typeof` comparisons
**ID:** `JS-0565` | **Lien:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0565)

![Minor](https://img.shields.io/badge/severity-minor-yellow) ![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

You should use the angular.isFunction method instead of the default JavaScript implementation (typeof function(){} ==="object Function").

**Rule based on Angular 1.x**


## Bad Practice

```javascript
// invalid
typeof someFunction === 'function' // error: You should use the angular.isFunction method
```

## Recommended

```javascript
// valid
angular.isFunction(someFunction);
```
