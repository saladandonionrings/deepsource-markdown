# Invalid ARIA state and/or property values
**ID:** `JS-0742` | **Lien:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0742)

![Minor](https://img.shields.io/badge/severity-minor-yellow) ![Bug Risk](https://img.shields.io/badge/type-bug_risk-green)

Adding `aria-hidden='true'` to an element removes that element and all of its children from the accessibility tree. This can improve the experience for assistive technology users by hiding:

* purely decorative content, such as icons or images
* duplicated content, such as repeated text
* offscreen or collapsed content, such as menus

Using `aria-hidden='false'` will not re-expose the element to assistive technology if any of its parents specify `aria-hidden='true'` .


## Bad Practice

```javascript
<!-- Bad: the aria-hidden state is of type true/false -->
<span aria-hidden="yes">foo</span>
```

## Recommended

```javascript
<!-- Good: the aria-hidden state is of type true/false -->
<span aria-hidden="true">foo</span>
```
