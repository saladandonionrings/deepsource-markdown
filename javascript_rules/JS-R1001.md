# Self import detected
**ID:** `JS-R1001` | **Link:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-R1001)

![Minor](https://img.shields.io/badge/severity-minor-yellow)![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

A module should never import itself. This usually happens as a mistake or typo and occurs mostly during refactoring. Self importing might result in unexpected results like wrong functions/variables being used.


## Bad Practice

```javascript
// <!-- calc.js -->
import calc from './calc.js'

export const sum = calc.sum
```

## Recommended

```javascript
// <!-- calc.js -->
import calc from './calc.polyfills.js'

export const sum = calc.sum
```
