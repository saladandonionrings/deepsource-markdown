# use `factory()` instead of `service()`
**ID:** `JS-0549` | **Lien:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0549)

![Minor](https://img.shields.io/badge/severity-minor-yellow) ![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

Prefer using `factory()` instead of `service()`

**Rule based on Angular 1.x**


## Bad Practice

```javascript
// invalid
angular.module('myModule').service('myService', function() {
    // ...
}); // error: You should prefer the factory() method instead of service()
```

## Recommended

```javascript
// valid
angular.module('myModule').factory('myService', function () {
    // ...
});

// valid
angular.module('myModule').value('someValue', {
    // ...
});
```
