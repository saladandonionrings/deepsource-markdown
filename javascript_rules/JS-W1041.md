# Found complex boolean return
**ID:** `JS-W1041` | **Link:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-W1041)

![Major](https://img.shields.io/badge/severity-major-orange)![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

The following pattern:


```javascript
if (condition) {
  return true
}
return false
```
can be refactored to:


```javascript
return condition
```
When `condition` is not a boolean value, it can be cast into one using the `Boolean` constructor.


## Bad Practice

```javascript
function isEven(num: number) {
  if (num % 2 === 0) return true
  return false
}

async function userExists(name: string) {
  if (await db.getUser(name)) return true
  return false
}
```

## Recommended

```javascript
function isEven(num: number) {
  return num % 2 === 0
}

async function userExists(name: string) {
  return Boolean(await db.getUser(name))
}
```
