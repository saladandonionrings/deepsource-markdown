# Invalid suffix in `@Directive`
**ID:** `JS-0598` | **Lien:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0598)

![Minor](https://img.shields.io/badge/severity-minor-yellow) ![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

Consistent conventions make it easy to quickly identify and reference assets of different types. Configure it using the following options :


```javascript
"directive-class-suffix": [true, "Directive", "MySuffix"]
```

## Bad Practice

```javascript
@Directive({
    selector: 'sgBarFoo'
})
class TestPage {}

// suffixes: ['Directive', 'View']
```

## Recommended

```javascript
@Directive({
    selector: 'sgBarFoo'
})
class TestDirective {}
```

```javascript
@Directive({
    selector: 'sgBarFoo'
})
class TestValidator implements Validator {}
```
