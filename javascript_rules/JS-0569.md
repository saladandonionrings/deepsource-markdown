# Prefer to use `$window` instead of `window`
**ID:** `JS-0569` | **Lien:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0569)

![Major](https://img.shields.io/badge/severity-major-orange) ![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

Instead of the default window object, prefer the AngularJS wrapper service $window.

**Rule based on Angular 1.x**


## Bad Practice

```javascript
window.alert('Hello world!');
```

## Recommended

```javascript
$window.alert('Hello world!');
```
