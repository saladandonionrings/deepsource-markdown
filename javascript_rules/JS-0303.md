# Consider using dot notation
**ID:** `JS-0303` | **Link:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0303)

![Minor](https://img.shields.io/badge/severity-minor-yellow)![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

In TypeScript, one can access properties using the dot notation (`object.property`) or square-bracket notation (`object["property"]`). However, the dot notation is often preferred because it is easier to read and less verbose.


## Bad Practice

```javascript
object["property"];
```

## Recommended

```javascript
object.property
```
