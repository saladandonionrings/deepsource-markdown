# Prefer shorthand form for react fragments
**ID:** `JS-0410` | **Lien:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0410)

![Minor](https://img.shields.io/badge/severity-minor-yellow) ![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

JSX Fragments can be written in two ways:

DeepSource recommends using the shorthand syntax over its alternative. This makes the code more concise and easier to read.


## Bad Practice

```javascript
<React.Fragment><Foo /></React.Fragment>
```

## Recommended

```javascript
<><Foo /></>
```
