# Too many arguments passed to function call
**ID:** `JS-W1038` | **Link:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-W1038)

![Minor](https://img.shields.io/badge/severity-minor-yellow)![Bug Risk](https://img.shields.io/badge/type-bug_risk-green)

JavaScript allows passing more arguments to a function than needed. However, doing so makes your code less readable, and may not produce the result you would expect from the function call.


## Bad Practice

```javascript
function mult(a, b, c) {
  return a * b * c;
}

mult(1, 2, 3, 4); // '4' will be ignored, and the result will be '6'.
mult(...nums); // this is OK.
```

## Recommended

```javascript
function mult(a, b, c) {
  return a * b * c;
}

mult(1, 2, 3);
mult(...nums); // this is OK.
```
