# Use `for-of` loop for array
**ID:** `JS-0361` | **Lien:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0361)

![Minor](https://img.shields.io/badge/severity-minor-yellow) ![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

A `for-of` loop is recommended when the loop index is only used to read from the collection.


## Bad Practice

```javascript
for (let i = 0; i < arr.length; i++) {
  console.log(arr[i]);
}
```

## Recommended

```javascript
for (const x of arr) {
  console.log(x);
}

for (let i = 0; i < arr.length; i++) {
  // i is used to write to arr, so for-of could not be used.
  arr[i] = 0;
}

for (let i = 0; i < arr.length; i++) {
  // i is used independent of arr, so for-of could not be used.
  console.log(i, arr[i]);
}
```
