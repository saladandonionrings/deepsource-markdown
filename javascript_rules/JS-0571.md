# Invalid lifecycle method in class
**ID:** `JS-0571` | **Link:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0571)

![Major](https://img.shields.io/badge/severity-major-orange)![Bug Risk](https://img.shields.io/badge/type-bug_risk-green)

Some lifecycle methods can only be used in certain class types.

* Class with`@Injectable`is not allowed when`ngAfterContentChecked()`,`ngAfterContentInit()`,`ngAfterViewInit()`,`ngDoCheck()`,`ngOnChanges()`,` ngOnInit()`methods are present.* Class with`@NgModule`is not allowed when`ngAfterContentChecked()`,`ngAfterContentInit()`,`ngAfterViewChecked()`,`ngAfterViewInit()`,`ngDoCheck()`,`ngOnChanges()`,`ngOnInit()`,`ngOnDestroy()`methods are present.* Class with`@Pipe`is not allowed when`ngAfterContentChecked()`,`ngAfterContentInit()`,`ngAfterViewChecked()`,`ngAfterViewInit()`,`ngDoCheck()`,`ngOnChanges()`,`ngOnInit()`methods are present.

## Bad Practice

```javascript
@Injectable()
        class Test {
            ngAfterContentChecked() { console.log('AfterContentChecked'); }
        }
```

```javascript
@NgModule()
        class Test {
          ngAfterViewChecked() { console.log('ngAfterViewChecked'); }
        }
```

## Recommended

```javascript
@Injectable()
     class Test {
      ngOnDestroy() { console.log('OnDestroy'); }
     }
```

```javascript
@Component()
    class Test {
        ngAfterContentChecked() { console.log('AfterContentChecked'); }
    }
```

```javascript
@Directive()
    class Test {
      ngOnInit() { console.log('OnInit'); }
    }
```

```javascript
@Pipe()
    class Test {
      ngOnDestroy() { console.log('OnDestroy'); }
    }
```
