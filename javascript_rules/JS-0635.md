# Found invalid `v-if` directives
**ID:** `JS-0635` | **Link:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0635)

![Major](https://img.shields.io/badge/severity-major-orange)![Bug Risk](https://img.shields.io/badge/type-bug_risk-green)

`v-if` directive is not valid :

* The directive has an argument.* The directive has a modifier.* The directive does not have an attribute value.* The directive is on the elements which have`v-else`/`v-else-if`directives.

## Bad Practice

```javascript
<!-- require no argument -->
<div v-if:aaa></div>

<!-- require no modifier -->
<div v-if.bbb></div>

<!-- found no attribute value -->
<div v-if></div>

<!-- 'v-else' and 'v-if' directives can't exist on the same element -->
<div v-else v-if="condition_one"></div>
```

## Recommended

```javascript
<!-- remove argument -->
<div v-if></div>

<!-- remove modifier -->
<div v-if></div>

<!-- add attribute value -->
<div v-if="condition_one"></div>

<!-- 'v-else' and 'v-if' directives can't exist on the same element -->
<div v-if="condition_one"></div>
<div v-else />
```
