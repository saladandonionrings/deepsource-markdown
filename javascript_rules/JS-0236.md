# Found unnecessary computed property keys in object literals
**ID:** `JS-0236` | **Lien:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0236)

![Minor](https://img.shields.io/badge/severity-minor-yellow) ![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

It's unnecessary to use computed properties with literals such as:


```javascript
var foo = {["a"]: "b"};
```
The code can be rewritten as:


```javascript
var foo = {"a": "b"};
```

## Bad Practice

```javascript
var a = { ['0']: 0 };
var a = { ['0+1,234']: 0 };
var a = { [0]: 0 };
var a = { ['x']: 0 };
var a = { ['x']() {} };
```

## Recommended

```javascript
var c = { 'a': 0 };
var c = { 0: 0 };
var a = { x() {} };
var c = { a: 0 };
var c = { '0+1,234': 0 };
```
