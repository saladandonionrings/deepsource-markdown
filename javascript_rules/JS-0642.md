# Found invalid `v-text` directives
**ID:** `JS-0642` | **Lien:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0642)

![Major](https://img.shields.io/badge/severity-major-orange) ![Bug Risk](https://img.shields.io/badge/type-bug_risk-green)

`v-text` is used to update the element’s [textContent](https://developer.mozilla.org/en-US/docs/Web/API/Node/textContent) . It is recommended to use `{{ Mustache }}` interpolations if an application needs to update the part of textContent.

`v-text` directive is invalid when:

* The directive has an argument.
* The directive has a modifier.
* The directive doesn't have an attribute value.


## Bad Practice

```javascript
<!-- 'v-text' directives require attribute value -->
<div v-text></div>

<!-- 'v-text' directives require no argument -->
<div v-text:aaa></div>

<!-- 'v-text' directives require no modifier -->
<div v-text.bbb></div>
```

## Recommended

```javascript
<!-- 'v-text' directives have attribute value -->
<div v-text="attribute"></div>

<!-- 'v-text' directives have no argument -->
<div v-text="attribute_one"></div>

<!-- 'v-text' directives have no modifier -->
<div v-text="attribute_two"></div>
```
