# Disallowed prefix for input names
**ID:** `JS-0584` | **Link:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0584)

![Minor](https://img.shields.io/badge/severity-minor-yellow)![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

HTML attributes are not prefixed. It's considered best not to prefix Inputs.

**Example**: `enabled` is preferred over `isEnabled`.


## Bad Practice

```javascript
import { Input } from '@angular/core';
@Component()
class TestComponent {
    @Input() is: string;
}
```

## Recommended

```javascript
import { Input } from '@angular/core';
@Component()
class TestComponent {
    @Input() label: string;
}
```
