# The usage of `__proto__` property is not recommended
**ID:** `JS-0084` | **Lien:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0084)

![Minor](https://img.shields.io/badge/severity-minor-yellow) ![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

`__proto__` property has been deprecated as of ECMAScript 3.1 and shouldn't be used. Use `Object.getPrototypeOf` and `Object.setPrototypeOf` instead.


## Bad Practice

```javascript
let objProto = obj.__proto__;
objProto = obj["__proto__"];

obj.__proto__ = proto;
obj["__proto__"] = proto;
```

## Recommended

```javascript
const proto = Object.getPrototypeOf(obj);
Object.setPrototypeOf(obj, proto);
```
