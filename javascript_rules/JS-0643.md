# Avoid using deprecated lifecycle hooks
**ID:** `JS-0643` | **Link:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0643)

![Major](https://img.shields.io/badge/severity-major-orange)![Bug Risk](https://img.shields.io/badge/type-bug_risk-green)

This issue is raised for the usage of `beforeDestroy()` and `destroyed()` hooks, which have been deprecated from Vue3 after the release of VueJS 3.x and are replaced by `beforeUnmount()` and `unmounted()`, respectively.


## Bad Practice

```javascript
beforeDestroy () {},
  destroyed () {}
```

## Recommended

```javascript
beforeUnmount () {},
  unmounted () {}
```
