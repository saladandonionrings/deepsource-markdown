# Prefer to have lang prop in elements
**ID:** `JS-0748` | **Lien:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0748)

![Minor](https://img.shields.io/badge/severity-minor-yellow) ![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)


## Bad Practice

```javascript
<html>
```

## Recommended

```javascript
<html lang="en">
<html lang="en-US">
<html lang={language}>
```
