# Missing `@Pipe` decorator
**ID:** `JS-0594` | **Link:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0594)

![Minor](https://img.shields.io/badge/severity-minor-yellow)![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

Ensure that classes implementing the `PipeTransform` interface use `@Pipe` decorator. Use the `@Pipe` annotation to declare that a given class is a pipe. The `@Pipe` decorator allows you to define the pipe name that is globally available for use in any template in the Angular app.


## Bad Practice

```javascript
@Test()
export class Test implements PipeTransform {}
```

## Recommended

```javascript
@Pipe({ name: 'test' })
export class TestPipe implements PipeTransform {
    transform(value: string) {}
}
```
