# Disallow unused properties
**ID:** `JS-0715` | **Lien:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0715)

![Minor](https://img.shields.io/badge/severity-minor-yellow) ![Bug Risk](https://img.shields.io/badge/type-bug_risk-green)

Unused properties should be eliminated.


## Bad Practice

```javascript
<!-- (`count` property not used) -->
<template>
  <div>{{ cnt }}</div>
</template>
<script>
  export default {
    props: ['count']
  }
</script>
```

## Recommended

```javascript
<template>
  <div>{{ count }}</div>
</template>
<script>
  export default {
    props: ['count']
  }
</script>
```
