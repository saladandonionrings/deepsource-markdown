# Disallow unsupported Vue.js syntax on the specified version
**ID:** `JS-0714` | **Lien:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0714)

![Major](https://img.shields.io/badge/severity-major-orange) ![Bug Risk](https://img.shields.io/badge/type-bug_risk-green)

There has been many versions of VueJS i.e. 2.5, 2.6, and 3.0 and each version has many changes that are not part of specified versions We recommend to use the VueJS Syntax according to valid version of VueJS.


## Bad Practice

```javascript
// v2.6.0

  <!-- argument on `v-model` -->
  <MyInput v-model:foo="val" />
  <!-- custom modifiers on `v-model` -->
  <MyComp v-model.foo.bar="text" />

// v2.5.0
<template>
  <!-- dynamic directive arguments -->
  <CustomComponent :[foo]="val" />
  <ListComponent>
    <!-- v-slot -->
    <template v-slot:name="props">
      {{ props.title }}
    </template>
    <template #name="props">
      {{ props.title }}
    </template>
  </ListComponent>
</template>
```

## Recommended

```javascript
// v2.6.0

<MyInput v-bind:foo.sync="val" />

// v2.5.0

<template>
  <CustomComponent :foo="val" />
  <ListComponent>
    <template slot="name" slot-scope="props">
      {{ props.title }}
    </template>
  </ListComponent>
```
