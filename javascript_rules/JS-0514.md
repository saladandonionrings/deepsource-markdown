# Use reference modules with the getter syntax
**ID:** `JS-0514` | **Lien:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0514)

![Minor](https://img.shields.io/badge/severity-minor-yellow) ![Style](https://img.shields.io/badge/type-style-blue)

When using a module, avoid using a variable and instead use chaining with the getter syntax. It produces readable code and avoids variable collisions or leaks.


## Bad Practice

```javascript
app.controller('MyController', function () {
    // ...
});
```

## Recommended

```javascript
angular.module('myModule').controller('MyController', function () {
    // ...
});
```
