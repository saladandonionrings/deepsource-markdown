# Specify consistent use of `$scope.digest()` or `$scope.apply()`
**ID:** `JS-0552` | **Link:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0552)

![Minor](https://img.shields.io/badge/severity-minor-yellow)![Performance](https://img.shields.io/badge/type-performance-white)

For the execution of the watchers, the $digest method will start from the scope in which we call the method. This will cause an performance improvement comparing to the $apply method, who start from the $rootScope.

**Rule based on Angular 1.x**


## Bad Practice

```javascript
// invalid
$scope.$digest(); // error: Instead of using the $digest() method, you should prefer $apply()
```

## Recommended

```javascript
// valid
$scope.$apply(function() {
    // ...
});
```
