# Require a name property in Vue components
**ID:** `JS-0720` | **Lien:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0720)

![Minor](https://img.shields.io/badge/severity-minor-yellow) ![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

Name property in a SFC module is mostly a convenience for debugging. It's also helpful for recursion.


## Bad Practice

```javascript
<script>
export default {
}

<script>
export default {
  notName: 'OurButton'
}
</script>

</script>
```

## Recommended

```javascript
<script>
export default {
  name: 'OurButton'
}
</script>
```
