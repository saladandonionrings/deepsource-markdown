# Consider Using `PascalCase` for user-defined JSX components
**ID:** `JS-0426` | **Lien:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0426)

![Minor](https://img.shields.io/badge/severity-minor-yellow) ![Style](https://img.shields.io/badge/type-style-blue)

It is recommended to use PascalCase for user-defined React JSX components since it uses the upper vs. lower case convention to distinguish between local component classes and HTML tags.


## Bad Practice

```javascript
<Test_component />

<TEST_COMPONENT />
```

## Recommended

```javascript
<div />

<TestComponent />

<TestComponent>
  <div />
</TestComponent>

<CSSTransitionGroup />
```
