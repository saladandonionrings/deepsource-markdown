# Found duplicate routing paths
**ID:** `JS-W1002` | **Link:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-W1002)

![Minor](https://img.shields.io/badge/severity-minor-yellow)![Bug Risk](https://img.shields.io/badge/type-bug_risk-green)

The issue is raised when Ember router has duplicate paths or whenever it has duplicate nested paths. The issue is fixed by renaming the paths of duplicate path names.


## Bad Practice

```javascript
this.route('main', { path: '/' }, function () {
  this.route('nested');
});
this.route('nested');
```

## Recommended

```javascript
this.route('main', { path: '/' }, function () {
  this.route('nested');
});
this.route('anotherNested');
```
