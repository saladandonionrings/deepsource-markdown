# Avoid using deprecated template lifecycle callback assignments
**ID:** `JS-0728` | **Link:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0728)

![Minor](https://img.shields.io/badge/severity-minor-yellow)![Bug Risk](https://img.shields.io/badge/type-bug_risk-green)

Assigning lifecycle callbacks to template properties has been deprecated in favor of the more robust template lifecycle callback registration functions.

Add `onRendered`, `onCreated`, and `onDestroyed` methods to `Template`. Assignments to `Template.card.rendered` are deprecated but are still supported for backward compatibility.


## Bad Practice

```javascript
Template.card.created = function { /* .. */ }
Template.card.rendered = function { /* .. */ }
Template.card.destroyed = function { /* .. */ }

Template[link].created = function { /* .. */ }
Template[link].rendered = function { /* .. */ }
Template[link].destroyed = function { /* .. */ }
```

## Recommended

```javascript
Template.card.onCreated(function { /* .. */ })
Template.card.onRendered(function { /* .. */ })
Template.card.ondestroyed(function { /* .. */ })

Template[card].onCreated(function { /* .. */ })
Template[card].onRendered(function { /* .. */ })
Template[card].ondestroyed(function { /* .. */ })
```
