# Disallow unnecessary `v-bind` directives
**ID:** `JS-0717` | **Link:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0717)

![Minor](https://img.shields.io/badge/severity-minor-yellow)![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

The `v-bind` with a string literal value can be changed to a static attribute definition.


## Bad Practice

```javascript
<div v-bind:foo="'bar'"/>
  <div :foo="'bar'"/>
```

## Recommended

```javascript
<div foo="bar"/>
  <div :foo="bar"/>
```
