# Found mustaches in `<textarea>`
**ID:** `JS-0617` | **Link:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0617)

![Minor](https://img.shields.io/badge/severity-minor-yellow)![Bug Risk](https://img.shields.io/badge/type-bug_risk-green)

Interpolation on textareas (`<textarea></textarea>`) won't work because it is not rendered by the Vue Compiler. It is recommended to use `v-model` instead.


## Bad Practice

```javascript
<template>
  <textarea>{{ message }}</textarea>
</template>
```

## Recommended

```javascript
<template>
  <textarea v-model="message" />
</template>
```
