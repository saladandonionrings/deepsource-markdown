# Found octal literals
**ID:** `JS-0081` | **Lien:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0081)

![Minor](https://img.shields.io/badge/severity-minor-yellow) ![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

Octal literals are numerals that begin with a leading zero, such as:


```javascript
const num = 071;      // 57
```
The leading zero which identifies an octal literal has been a source of confusion and error in JavaScript code. ECMAScript 5 deprecates the use of octal numeric literals.


## Bad Practice

```javascript
const num = 071;
const result = 5 + 07;
```

## Recommended

```javascript
const num  = "071";
```
