# Use `v-bind:is` in `<component>` elements
**ID:** `JS-0620` | **Lien:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0620)

![Minor](https://img.shields.io/badge/severity-minor-yellow) ![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

In Vue, `v-bind` lets you bind an HTML attribute to a JavaScript expression but there are some HTML elements, such as `<ul>` , `<ol>` , `<table>` and `<select>` have restrictions on what elements can appear inside them.

The workaround is to use the `is` special attribute.


## Bad Practice

```javascript
<template>
  <component/>
  <component is="type"/>
</template>
```

## Recommended

```javascript
<template>
  <component :is="type"/>
  <component v-bind:is="type"/>
</template>
```
