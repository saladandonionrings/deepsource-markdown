# Prefer that `autoFocus` prop is not used on elements
**ID:** `JS-0757` | **Lien:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0757)

![Minor](https://img.shields.io/badge/severity-minor-yellow) ![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

The issue enforces that the `autoFocus` prop is not used on elements because autofocusing elements can cause usability issues for sighted and non-sighted users alike.


## Bad Practice

```javascript
<div autoFocus />
<div autoFocus="true" />
<div autoFocus="false" />
<div autoFocus={undefined} />
```

## Recommended

```javascript
<div />
```
