# Detected aliasing this
**ID:** `JS-0342` | **Lien:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0342)

![Major](https://img.shields.io/badge/severity-major-orange) ![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

This rule prohibits assigning variables to `this` .


## Bad Practice

```javascript
//Assigning a variable to this instead of properly using arrow lambdas may be a symptom of pre-ES6 practices or not managing scope well.

//Instead of storing a reference to this and using it inside a function () {:

const self = this;

setTimeout(function () {
  self.doWork();
});
```

## Recommended

```javascript
// Use () => arrow lambdas, as they preserve this scope for you:

setTimeout(() => {
  this.doWork();
});
```
