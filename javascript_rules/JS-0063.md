# Unnecessary labels
**ID:** `JS-0063` | **Link:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0063)

![Minor](https://img.shields.io/badge/severity-minor-yellow)![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

If a loop contains no nested loops or switches, labeling the loop is unnecessary.


```javascript
A: while (a) {
    break A;
}
```
You can achieve the same result by removing the label and using break or continue without a label.


## Bad Practice

```javascript
A: while (a) {
    break A;
}

B: for (let i = 0; i < 10; ++i) {
    break B;
}

C: switch (a) {
    case 0:
        break C;
}
```

## Recommended

```javascript
while (a) {
    console.log(a);
    if (a == 0) break;
    --a;
}

for (let i = 0; i < 10; ++i) {
    // code
    break;
}

switch (a) {
    case 0:
        break;
}
```
