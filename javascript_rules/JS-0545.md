# Require a consistent DI syntax
**ID:** `JS-0545` | **Lien:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0545)

![Minor](https://img.shields.io/badge/severity-minor-yellow) ![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

All your DI should use the same syntax : the Array, function, or `$inject` syntaxes.

**Rule based on Angular 1.x**


## Bad Practice

```javascript
// invalid
angular.module('myModule').factory('myService', ['$http', '$log', 'anotherService', function ($http, $log, anotherService) {
    // ...
}]); // error: You should use the function syntax for DI
```

## Recommended

```javascript
// valid
angular.module('myModule').factory('myService', function ($http, $log, anotherService) {
   // ...
});
```
