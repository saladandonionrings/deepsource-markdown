# Avoid adding stylesheet manually
**ID:** `JS-W1021` | **Link:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-W1021)

![Minor](https://img.shields.io/badge/severity-minor-yellow)![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

Using an HTML `link` element to link to an external stylesheet can affect the resource loading (CSS) on your web page. It is recommended to use the Next.js built-in CSS or stylesheet loading instead, which gives you the ability to import CSS from any JavaScript file.


## Bad Practice

```javascript
import { Head } from "next/document";
export class NavBar extends Head {
  render() {
    return (
      <div>
        <h1>Welcome to DeepSource!</h1>
        <link href="/_next/static/css/styles.css" rel="stylesheet" />
      </div>
    );
  }
}
```

## Recommended

```javascript
import { Head } from "next/document";
import styles from './styles.css'
export class NavBar extends Head {
  render() {
    return (
      <div>
        <h1 className={styles.Heading}>Welcome to DeepSource!</h1>
      </div>
    );
  }
}
```
