# Invalid use of test waiters
**ID:** `JS-0792` | **Link:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0792)

![Minor](https://img.shields.io/badge/severity-minor-yellow)![Bug Risk](https://img.shields.io/badge/type-bug_risk-green)

Test waiters should follow follwing best practices:

* Used in module scope* Assigned to a variable

## Bad Practice

```javascript
import { buildWaiter } from 'ember-test-waiters';

function useWaiter() {
  const myOtherWaiter = buildWaiter('the second'); // inside function
}
```

```javascript
import { buildWaiter } from 'ember-test-waiters';

buildWaiter('the second'); // not stored in variable
```

## Recommended

```javascript
import { buildWaiter } from 'ember-test-waiters';

const myWaiter = buildWaiter('waiterName');
```
