# Avoid direct use of `this` in controllers
**ID:** `JS-0519` | **Lien:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0519)

![Major](https://img.shields.io/badge/severity-major-orange) ![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

The `this` keyword is contextual. Using it within a function inside a controller may change its context. Using another variable to capture the context of `this` will fix the problem.


## Bad Practice

```javascript
angular
    .module('test')
    .controller('TestController', function() {
        this.test = 'test';
    });
```

## Recommended

```javascript
angular
    .module('test')
    .controller('TestController', function() {
        let vm = this;
        vm.test = 'test';
    });
```
