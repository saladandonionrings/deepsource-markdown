# React must be present in scope when using JSX
**ID:** `JS-0464` | **Lien:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0464)

![Minor](https://img.shields.io/badge/severity-minor-yellow) ![Bug Risk](https://img.shields.io/badge/type-bug_risk-green)

The `React` object contains many objects that are referenced indirectly when expanding JSX.

When using JSX, `<a />` expands to `React.createElement("a")` . Therefore `React` must be imported.


## Bad Practice

```javascript
const Hello = <div>Hello {this.props.name}</div>;
```

## Recommended

```javascript
import React from 'react';
const Hello = <div>Hello {this.props.name}</div>;
```
