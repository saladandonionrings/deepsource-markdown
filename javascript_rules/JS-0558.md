# Consider using `ofangular.fromJson` and `angular.toJson`
**ID:** `JS-0558` | **Lien:** [DeepSource](https://deepsource.com/directory/javascript/issues/JS-0558)

![Minor](https://img.shields.io/badge/severity-minor-yellow) ![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

You should use angular.fromJson or angular.toJson instead of JSON.parse and JSON.stringify

**Rule based on Angular 1.x**


## Bad Practice

```javascript
// invalid
JSON.stringify({
    // ...
}); // error: You should use the angular.toJson method instead of JSON.stringify

// invalid
var data = JSON.parse('{"message": "Hello World!"}'); // error: You should use the angular.fromJson method instead of JSON.parse
```

## Recommended

```javascript
// valid
angular.toJson({
    // ...
});

// valid
var data = angular.fromJson('{"message": "Hello World!"}');
```
