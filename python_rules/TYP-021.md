# Inconsistent use of `type annotation` and `type comment`
**ID:** `TYP-021` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/TYP-021)

![Major](https://img.shields.io/badge/severity-major-orange) ![Type Check](https://img.shields.io/badge/type-type_check-lightgrey)

[PEP 484](https://www.python.org/dev/peps/pep-0484/) disallows both type annotations and type comments. Please use type annotation for python 3 and type comments for python 2.
