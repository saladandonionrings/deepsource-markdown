# Not enough arguments in format string
**ID:** `PYL-E1306` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PYL-E1306)

![Critical](https://img.shields.io/badge/severity-critical-red) ![Bug Risk](https://img.shields.io/badge/type-bug_risk-green)

Format string do not contain enough arguments to unpack. This can cause an error.


## Bad practice

```python
print('Hey {}, {} messaged you'.format('Bob'))
```

## Recommended

```python
print('Hey {}, {} messaged you'.format('Bob', 'Alex'))
```
