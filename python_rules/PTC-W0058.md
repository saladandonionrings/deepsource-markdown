# `list` object used during set creation
**ID:** `PTC-W0058` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PTC-W0058)

![Critical](https://img.shields.io/badge/severity-critical-red) ![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

`list` type objects are unhashable and thus cannot be used for set initialization. Use `tuple` instead.


## Not preferred:

```python
myset = {[1, 2, 3]}
set_b = set([[1, 2], ["a", "b"]])
set_c = frozenset([(1, 2), ["a", "b"]])
```

## Preferred:

```python
set_a = {(1, 2, 3)}
set_b = set([(1, 2), ("a", "b")])
set_c = frozenset([(1, 2), ("a", "b")])
```
