# Consider iterating dictionary
**ID:** `PYL-C0201` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PYL-C0201)

![Major](https://img.shields.io/badge/severity-major-orange) ![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

Consider iterating the dictionary directly instead of calling `.keys()` . Using `for key in dictionary` would always iterate the dictionary keys.


## Bad practice

```python
sounds = {"cow": "moo", "dog": "bark"}
for key in sounds.keys():
    print(key)
```

## Recommended

```python
sounds = {"cow": "moo", "dog": "bark"}
for key in sounds:
    print(key)
```
