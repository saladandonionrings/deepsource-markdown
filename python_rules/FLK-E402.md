# Module level import not at the top of the file
**ID:** `FLK-E402` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/FLK-E402)

![Minor](https://img.shields.io/badge/severity-minor-yellow) ![Style](https://img.shields.io/badge/type-style-blue)

All module level imports should be at the top of the file. This means that there should be no statements in between module level imports.
