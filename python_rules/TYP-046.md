# Inconsistent number or arguments detected for type hint
**ID:** `TYP-046` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/TYP-046)

![Major](https://img.shields.io/badge/severity-major-orange) ![Type Check](https://img.shields.io/badge/type-type_check-lightgrey)

Inconsistent number of argument given to a type hint. For example, `Optional[...]` , and `Type[...]` must have exactly one type argument. Please see the error message for more details.
