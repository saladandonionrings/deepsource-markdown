# Unnecessary `pass` statement
**ID:** `PYL-W0107` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PYL-W0107)

![Minor](https://img.shields.io/badge/severity-minor-yellow) ![Style](https://img.shields.io/badge/type-style-blue)

The `pass` statement used here is not necessary. You can safely remove this.


```python
class Docstring:
    '''This is a dummy class with docstring.'''
    pass
```

```python
class Docstring:
    '''This is a dummy class with docstring.'''
```
