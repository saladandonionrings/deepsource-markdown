# Use of a deprecated module detected
**ID:** `PYL-W0402` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PYL-W0402)

![Major](https://img.shields.io/badge/severity-major-orange) ![Bug Risk](https://img.shields.io/badge/type-bug_risk-green)

The module imported has been deprecated for the current version of Python.

Deprecated modules no longer get new feature or security updates, and are usually removed from the Python standard library 3-5 years after deprecation. It is highly recommended to stop depending on deprecated built-in modules.
