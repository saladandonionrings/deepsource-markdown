# Wrong use of `Final` type
**ID:** `TYP-045` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/TYP-045)

![Major](https://img.shields.io/badge/severity-major-orange) ![Type Check](https://img.shields.io/badge/type-type_check-lightgrey)

Wrong use of `Final[...]` . Final can be only used as an outermost qualifier in a variable annotation. You can read more about using `Final` [here](https://mypy.readthedocs.io/en/latest/final_attrs.html#details-of-using-final) .
