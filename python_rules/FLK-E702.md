# Multiple statements detected on one line
**ID:** `FLK-E702` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/FLK-E702)

![Minor](https://img.shields.io/badge/severity-minor-yellow) ![Style](https://img.shields.io/badge/type-style-blue)

Multiple statements should not be on the same line separated by a semicolon. It is considered a good practice to have statements on their own separate lines. This makes the code more readable.
