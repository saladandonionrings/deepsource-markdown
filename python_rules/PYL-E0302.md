# Special method defined with an invalid signature
**ID:** `PYL-E0302` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PYL-E0302)

![Major](https://img.shields.io/badge/severity-major-orange) ![Bug Risk](https://img.shields.io/badge/type-bug_risk-green)

Special methods are a set of predefined methods you can use to enrich your classes. Here, one or more special methods has been defined with an invalid number of parameters. If the special method has too few or too many arguments, it might not work at all. It is therefore recommended to take another look at the method definition.


## Bad practice

```python
class Example:

    def __enter__(self, other):  # special method `__enter__` provided with an extra arg
        ...
```

## Recommended

```python
class Example:

    def __enter__(self):
        ...
```
