# Invalid format character
**ID:** `PYL-E1300` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PYL-E1300)

![Critical](https://img.shields.io/badge/severity-critical-red) ![Bug Risk](https://img.shields.io/badge/type-bug_risk-green)

An unsupported format character is used in the format string. It is recommended to check the format string once again and see if there is a need to escape some special character.


## Bad practice

```python
marks = 84
my_string = 'Marks: %d%!' % (marks,)
                    # ^ un-escaped `%` sign
```

## Recommended
Use two `%` signs to print an actual `%` character in a format string


```python
marks = 84
my_string = 'Marks: %d%%!' % (marks,)
print(my_string)  # Marks: 84%!
```
