# `print` statement detected
**ID:** `PYL-E1601` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PYL-E1601)

![Major](https://img.shields.io/badge/severity-major-orange) ![Bug Risk](https://img.shields.io/badge/type-bug_risk-green)

`print` is being used as a statement here. It has been converted to a function in Python 3. This will cause an error.


## Bad practice

```python
x = 1
print x
```

## Recommended

```python
x = 1
print(x)
```
