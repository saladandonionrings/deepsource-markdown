# Statement not reachable on execution
**ID:** `PYL-W0101` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PYL-W0101)

![Critical](https://img.shields.io/badge/severity-critical-red) ![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

This statement is unreachable, as the control flow will never reach upto this point. Consider removing this part of code or re-evaluating the control flow.


## Bad practice

```python
def count_pairs(users):
    pairs = 0
    for user in users:
        if user.type == "student":
            pairs += 1
    return pairs
    pairs /= 2  # <-- Unreachable
```

## Preferred

```python
def count_pairs(users):
    pairs = 0
    for user in users:
        if user.type == "student":
            pairs += 1
    pairs /= 2
    return pairs
```
