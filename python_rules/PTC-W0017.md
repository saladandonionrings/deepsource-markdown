# Redundant call
**ID:** `PTC-W0017` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PTC-W0017)

![Major](https://img.shields.io/badge/severity-major-orange) ![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

It is unnecessary to use, e.g., `list` around a list comprehension since it is equivalent without it. Same goes for `set` and `dict` comprehensions.


## Not preferred

```python
my_list = list([i for i in range(10)])
my_set = set({i for i in range(10)})
my_dict = dict({i: i*i for i in range(10)})
```

## Preferred

```python
my_list = [i for i in range(10)]
my_set = {i for i in range(10)}
my_dict = {i: i*i for i in range(10)}
```
