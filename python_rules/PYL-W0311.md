# Bad indentation detected
**ID:** `PYL-W0311` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PYL-W0311)

![Minor](https://img.shields.io/badge/severity-minor-yellow) ![Style](https://img.shields.io/badge/type-style-blue)

Statements use indentations that are not standard four spaces. [PEP 8](https://python.org/dev/peps/pep-0008) recommends 4 spaces to be used as single indent.
