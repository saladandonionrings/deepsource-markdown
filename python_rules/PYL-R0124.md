# Comparison with itself
**ID:** `PYL-R0124` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PYL-R0124)

![Major](https://img.shields.io/badge/severity-major-orange) ![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

Comparison of an object with itself always returns `True` . This is redundant.


## Bad practice

```python
if x >= x:
    y = square(x)

if y != y:
    y = 0

return y
```

## Recommended

```python
y = square(x)

return y
```
