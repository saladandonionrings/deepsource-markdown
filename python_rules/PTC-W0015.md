# Unnecessary generator
**ID:** `PTC-W0015` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PTC-W0015)

![Major](https://img.shields.io/badge/severity-major-orange) ![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

It is unnecessary to use `list` , `set` , `dict` around a generator expression to get an object of that type since there are comprehensions for these types.


## Bad practice

```python
squares = list(x**2 for x in range(1, 10))
large_numbers = set(n for n in numbers if n > 1000)
tree_counts = dict((tree, counts[tree]) for tree in trees)
```

## Recommended

```python
squares = [x**2 for x in range(1, 10)]
large_numbers = {n for n in numbers if n > 1000}
tree_counts = {tree: counts[tree] for tree in trees}
```
