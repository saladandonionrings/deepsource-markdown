# Unnecessary literal
**ID:** `PTC-W0018` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PTC-W0018)

![Minor](https://img.shields.io/badge/severity-minor-yellow) ![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

It is unnecessary to use a `list` or `tuple` literal within a call to `tuple` , `list` , `set` , or `dict` since there is a literal syntax for these types.
