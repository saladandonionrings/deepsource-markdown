# Format string key is not a string
**ID:** `PYL-W1300` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PYL-W1300)

![Major](https://img.shields.io/badge/severity-major-orange) ![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

Format string that uses named conversion specifiers is used with a dictionary whose keys are not all strings. A format string named conversion specifier is always a string. So, make sure all the dictionary keys are strings as well.


```python
def greet_user(user):
    print(
        "Hi %(1)s!. Welcome to %(location)s."
        % {1: user.first_name, "location": user.updated_loc}
    )
```

```python
def greet_user(user):
    print(
        "Hi %(name)s!. Welcome to %(location)s."
        % {"name": user.first_name, "location": user.updated_loc}
    )
```
