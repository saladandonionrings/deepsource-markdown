# Invalid return `type hint` for async generator function
**ID:** `TYP-010` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/TYP-010)

![Major](https://img.shields.io/badge/severity-major-orange) ![Type Check](https://img.shields.io/badge/type-type_check-lightgrey)

The return type of an async generator function should be `AsyncGenerator` or one of its supertypes.
