# `Final[...]` shall have at most one type argument
**ID:** `TYP-029` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/TYP-029)

![Major](https://img.shields.io/badge/severity-major-orange) ![Type Check](https://img.shields.io/badge/type-type_check-lightgrey)

More than one `Type` argument given to `Final[...]` . This is invalid. Read all about `Final` [here](https://mypy.readthedocs.io/en/latest/final_attrs.html) .
