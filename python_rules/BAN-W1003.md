# Audit required: Insecure hash function
**ID:** `BAN-W1003` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/BAN-W1003)

![Major](https://img.shields.io/badge/severity-major-orange) ![Security](https://img.shields.io/badge/type-security-red)

D2, MD4, MD5, SHA1 signature algorithms are known to be vulnerable to collision attacks. Attackers can exploit this to generate another certificate with the same digital signature, allowing them to masquerade as the affected service.
