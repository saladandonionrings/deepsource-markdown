# Unary operand used on an unsupported object
**ID:** `PYL-E1130` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PYL-E1130)

![Critical](https://img.shields.io/badge/severity-critical-red) ![Bug Risk](https://img.shields.io/badge/type-bug_risk-green)

A unary operand is used on an object which does not support this type of operation. This is an error.
