# Format string contains unused key
**ID:** `PYL-W1301` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PYL-W1301)

![Major](https://img.shields.io/badge/severity-major-orange) ![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

Format string is using named conversion specifiers is being used with a dictionary that contains keys not required by the format string. It is recommend to remove the unused keys.


```python
def greet_user(user):
    print(
        "Hi %(name)s!. Welcome to %(location)s."
        % {"name": user.first_name, "location": user.updated_loc, "age": user.age}
    )
```
Here, the key `age` is nowhere used in the format string and should be removed.


```python
def greet_user(user):
    print(
        "Hi %(name)s!. Welcome to %(location)s."
        % {"name": user.first_name, "location": user.updated_loc}
    )
```
