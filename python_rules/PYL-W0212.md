# Protected member accessed from outside the class
**ID:** `PYL-W0212` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PYL-W0212)

![Minor](https://img.shields.io/badge/severity-minor-yellow) ![Bug Risk](https://img.shields.io/badge/type-bug_risk-green)

Accessing a protected member (a member prefixed with `_` ) of a class from outside that class is not recommended, since the creator of that class did not intend this member to be exposed. If accesing this attribute outside of the class is absolutely needed, refactor it such that it becomes part of the public interface of the class.


## Bad practice

```python
class Rectangle(object):
    def __init__(self, height, width):
        self._height = height
        self._width = width

r = Rectangle(4, 8)
area = r._height * r._width
```

## Preferred:

```python
class Rectangle(object):
    def __init__(self, height, width):
        self._height = height
        self._width = width

    def area():
        return self._height * self._width

r = Rectangle(4, 8)
area = r.area()
```
