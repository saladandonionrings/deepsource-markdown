# `iter` returns a non-iterator
**ID:** `PYL-E0301` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PYL-E0301)

![Critical](https://img.shields.io/badge/severity-critical-red) ![Bug Risk](https://img.shields.io/badge/type-bug_risk-green)

The `__iter__` method of a class must return an iterator with a `__next__` method.


## Bad practice

```python
class MyIterator:
    def __iter__(self):
        return self
```

## Recommended

```python
class MyIterator:
    def __iter__(self):
        return self

    def __next__(self):
        return 42
```
