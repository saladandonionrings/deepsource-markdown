# Method has no argument
**ID:** `PYL-E0211` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PYL-E0211)

![Critical](https://img.shields.io/badge/severity-critical-red) ![Bug Risk](https://img.shields.io/badge/type-bug_risk-green)

A method which should have the bound instance as first argument has no argument defined. Python will throw an error when you'll try to call this method.


## Bad practice

```python
class Cls:
    def some_method():
        ...

    @classmethod
    def something():
        ...
```

## Recommended

```python
class Cls:
    def some_method(self):
        ...

    @classmethod
    def something(cls):
        ...
```
