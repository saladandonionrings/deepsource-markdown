# Positional arguments are out of order
**ID:** `PYL-W1114` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PYL-W1114)

![Minor](https://img.shields.io/badge/severity-minor-yellow) ![Style](https://img.shields.io/badge/type-style-blue)

Positional arguments (with the same name as function parameters) appear to be out of order. Ensure that the order of arguments passed is correct.


## Bad practice

```python
def func_with_two_args(arg_one, arg_two):
    ...

arg_one = 1
arg_two = 2
func_with_two_args(arg_two, arg_one)
```

## Recommended

```python
def func_with_two_args(arg_one, arg_two):
    ...

arg_one = 1
arg_two = 2
func_with_two_args(arg_one, arg_two)
```
