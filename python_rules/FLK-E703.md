# Statement ends with a semicolon
**ID:** `FLK-E703` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/FLK-E703)

![Minor](https://img.shields.io/badge/severity-minor-yellow) ![Style](https://img.shields.io/badge/type-style-blue)

Unlike programming languages like Java and C#, statements in Python do not need to end with a semicolon. They can be safely removed.
