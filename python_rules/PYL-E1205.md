# Logging format string contains too many arguments
**ID:** `PYL-E1205` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PYL-E1205)

![Critical](https://img.shields.io/badge/severity-critical-red) ![Bug Risk](https://img.shields.io/badge/type-bug_risk-green)

Too many arguments passed to the logging format string.


## Bad practice

```python
logger.error("Failed to retrieve URL ", url)
```

## Recommended

```python
logger.error("Failed to retrieve URL %s", url)
```
