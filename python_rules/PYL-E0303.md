# Invalid return for `__len__`
**ID:** `PYL-E0303` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PYL-E0303)

![Critical](https://img.shields.io/badge/severity-critical-red) ![Bug Risk](https://img.shields.io/badge/type-bug_risk-green)

The `__len__` method of a class must return a non-negative integer.


## Bad practice

```python
class BadLenOne:
    """ __len__ returns a negative integer """

    def __len__(self):
        return -1


class SecondBadLen(object):
    """ __len__ returns non-int """

    def __len__(self):
        return 3.0
```

## Recommended
This is valid:


```python
class BadLenOne:
    """ __len__ returns a non-negative integer """

    def __len__(self):
        return 1
```
