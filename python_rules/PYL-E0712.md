# Exception caught which does not inherit from `Exception`
**ID:** `PYL-E0712` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PYL-E0712)

![Critical](https://img.shields.io/badge/severity-critical-red) ![Bug Risk](https://img.shields.io/badge/type-bug_risk-green)

For an exception to be caught, it must inherit from the `Exception` class.


## Bad practice

```python
try:
    if foo < 5:
        print('foo is small')
except NotImplemented:  # This is not an error class!
    print('Some error occured')
```

## Recommended

## Bad practice

```python
try:
    if foo < 5:
        print('foo is small')
except NotImplementedError:  # This is a valid error class
    print('Some error occured')
```
