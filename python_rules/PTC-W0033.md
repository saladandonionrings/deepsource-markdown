# Formatted string used as docstring
**ID:** `PTC-W0033` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PTC-W0033)

![Major](https://img.shields.io/badge/severity-major-orange) ![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

Formatted string literals cannot be used as docstrings, even if they do not include expressions. If used, the formatted strings are just evaluated as an expression in the function body.

Docstrings in Python need to be a constant, which means that trying to format a docstring at runtime, using `.format()` or f-strings simply doesn't work.

For example:


```python
def foo():
    f'''This is intended to be the docstring for {__name__}'''
    ...
```
The value of `foo.__doc__` would be `None` here.
