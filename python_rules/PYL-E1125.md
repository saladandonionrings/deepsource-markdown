# Missing mandatory keyword argument in function call
**ID:** `PYL-E1125` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PYL-E1125)

![Critical](https://img.shields.io/badge/severity-critical-red) ![Bug Risk](https://img.shields.io/badge/type-bug_risk-green)

The function call does not pass a mandatory keyword-only argument. This is an error.


## Bad practice

```python
def func(foo, *args, test):
    print(foo, test)

func(10, 20, 30)  # Missing required keyword argument 'test'
```

## Recommended

```python
def func(foo, *args, test):
    print(foo, test)

func(10, 20, 30, test=False)
```
