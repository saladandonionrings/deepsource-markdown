# Audit required: Use of `md5`
**ID:** `BAN-B303` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/BAN-B303)

![Minor](https://img.shields.io/badge/severity-minor-yellow) ![Security](https://img.shields.io/badge/type-security-red)

Use of insecure MD2, MD4, MD5, or SHA1 hash functions should be avoided. Using more secure algorithms like SHA256 or SHA512.
