# Unsupported binary operation
**ID:** `PYL-E1131` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PYL-E1131)

![Critical](https://img.shields.io/badge/severity-critical-red) ![Bug Risk](https://img.shields.io/badge/type-bug_risk-green)

Binary operation being done between operands that do not support it. This can cause an error on execution.
