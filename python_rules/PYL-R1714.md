# Consider using `in`
**ID:** `PYL-R1714` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PYL-R1714)

![Major](https://img.shields.io/badge/severity-major-orange) ![Performance](https://img.shields.io/badge/type-performance-white)

To check if a variable is equal to one of many values, combine the values into a tuple and check if the variable is contained `in` it instead of checking for equality against each of the values. This is faster, less verbose, and more readable.


## Bad practice

```python
if x == 1 or x == 2 or x == 3:
    do_something()
```

## Recommended

```python
if x in (1, 2, 3):
    do_something()
```
