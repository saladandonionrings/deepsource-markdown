# Explicitly declared types are required for all `Protocol` members
**ID:** `TYP-031` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/TYP-031)

![Major](https://img.shields.io/badge/severity-major-orange) ![Type Check](https://img.shields.io/badge/type-type_check-lightgrey)

All members of a protocol are required to have explicitly declared types. You can read more about it in this [PEP](https://www.python.org/dev/peps/pep-0544/)
