# Redundant `content_type` parameter for `JsonResponse()` detected
**ID:** `PYL-R5103` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PYL-R5103)

![Major](https://img.shields.io/badge/severity-major-orange) ![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

`JsonResponse()` contains `content_type` parameter. This is either redundant or the content_type is not JSON, which is probably an error.
