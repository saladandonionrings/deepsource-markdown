# Unnecessary use of comprehension
**ID:** `PYL-R1721` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PYL-R1721)

![Major](https://img.shields.io/badge/severity-major-orange) ![Performance](https://img.shields.io/badge/type-performance-white)

It is unnecessary to use a comprehension just to loop over the `iterable` and create a `list` / `set` / `dict` out of it. Python has a specialized set of tools for this task: the `list` / `set` / `dict` constructors, which are faster and more readable.


## Bad practice

```python
states = [
    ('AL', 'Alabama'),
    ('AK', 'Alaska'),
    ('AZ', 'Arizona'),
    ('AR', 'Arkansas'),
    ('CA', 'California'),
    # ...
]

abbreviations_to_names = {
    abbreviation: name
    for abbreviation, name in states
}
```

## Recommended

```python
states = [
    ('AL', 'Alabama'),
    ('AK', 'Alaska'),
    ('AZ', 'Arizona'),
    ('AR', 'Arkansas'),
    ('CA', 'California'),
    # ...
]

abbreviations_to_names = dict(states)
```
