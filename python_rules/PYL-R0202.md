# No `@classmethod` decorator
**ID:** `PYL-R0202` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PYL-R0202)

![Major](https://img.shields.io/badge/severity-major-orange) ![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

It is recommended to use the decorator syntax to declare a classmethod. Read more about classmethod [here](https://docs.python.org/3/library/functions.html#classmethod)


## Not preferred

```python
class MyClass:

    def cmethod(cls):
        '''class method-to-be'''

    # Convert cmethod to `classmethod`
    cmethod = classmethod(cmethod)
```

## Preferred

```python
class MyClass:

    @classmethod
    def cmethod(cls):
        '''class method-to-be'''
```
