# Visually indented line with same indent as next logical line
**ID:** `FLK-E129` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/FLK-E129)

![Minor](https://img.shields.io/badge/severity-minor-yellow) ![Style](https://img.shields.io/badge/type-style-blue)

A visual indented line has the same indentation as the next logical line. This can make it hard to read.
