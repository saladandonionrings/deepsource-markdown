# Consider using `all`
**ID:** `PY-W0075` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PY-W0075)

![Major](https://img.shields.io/badge/severity-major-orange) ![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

It is recommended to use `all` for this loop, as it will make your code smaller and more readable.


## Bad practice

```python
def all_even(items):
    for item in items:
        if item % 2 != 0:
            return False
    return True
```

## Recommended

```python
def all_even(items):
    return all(item % 2 == 0 for item in items)
```
