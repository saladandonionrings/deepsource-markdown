# Missing format attribute
**ID:** `PYL-W1306` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PYL-W1306)

![Major](https://img.shields.io/badge/severity-major-orange) ![Bug Risk](https://img.shields.io/badge/type-bug_risk-green)

The format string uses an attribute specifier like `{0.length}` , but the argument passed for formatting doesn't have that attribute. This is an error.

Note: String formatting uses a limited Python-like syntax. Calls are not supported in this syntax; only subscription (indexing by number or unquoted (!) name) and attribute access is supported.


## Bad practice

```python
class API:
    endpoint = 'example.com/api'

api = API()
print('{0.endopint}'.format(api))  # Misspelled property `endpoint`
```

## Recommended

```python
class API:
    endpoint = 'example.com/api'

api = API()
print('{0.endpoint}'.format(api))
```
