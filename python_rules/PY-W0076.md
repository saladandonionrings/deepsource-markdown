# Assigning to an attribute not defined in class `__slots__`
**ID:** `PY-W0076` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PY-W0076)

![Critical](https://img.shields.io/badge/severity-critical-red) ![Bug Risk](https://img.shields.io/badge/type-bug_risk-green)

Assigning to an attribute which is not defined in the class slots will raise an error. It is recommended to add the name in `__slots__` .


## Bad practice

```python
class E:
    __slots__ = ["foo"]

    def __init__(self):
        self.not_foo = "Not foo"
        self.not_bar = "Not Bar"
```

## Recommended

```python
class E:
    __slots__ = ["foo", "not_foo", "not_bar"]

    def __init__(self):
        self.not_foo = "Not foo"
        self.not_bar = "Not Bar"
```
Note: During Autofix, DeepSource will add the missing names in `slots` .
