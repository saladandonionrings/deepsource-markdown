# No `@staticmethod` decorator
**ID:** `PYL-R0203` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PYL-R0203)

![Major](https://img.shields.io/badge/severity-major-orange) ![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

It is recommended to use the decorator syntax to declare a staticmethod. Read more about staticmethod [here](https://docs.python.org/3/library/functions.html#staticmethod)


## Not preferred

```python
class MyClass:

    def smethod():
        '''static method-to-be'''

    # Convert smethod to `staticmethod`
    smethod = staticmethod(smethod)
```

## Preferred

```python
class MyClass:

    @staticmethod
    def smethod():
        '''static method-to-be'''
```
