# Name used prior global declaration
**ID:** `PYL-E0118` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PYL-E0118)

![Critical](https://img.shields.io/badge/severity-critical-red) ![Bug Risk](https://img.shields.io/badge/type-bug_risk-green)

A global variable is being accessed in a local scope before it has been defined with a `global` statement. Doing so will raise a `SyntaxError` .


## Bad practice

```python
var = "Some Value"


def foo():
    var = "New value"
    global var
    ...
```

## Preferred

```python
var = "Some Value"


def foo():
    global var
    var = "New value"
    ...
```
