# Format string contains excess arguments
**ID:** `PYL-E1305` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PYL-E1305)

![Critical](https://img.shields.io/badge/severity-critical-red) ![Bug Risk](https://img.shields.io/badge/type-bug_risk-green)

Format string has been given excess variables to unpack. This can cause an error.


## Bad practice

```python
print('Hey Bob,{} sent you a message'.format('Bob', 'Alex'))
```

## Recommended

```python
print('Hey {},{} sent you a message'.format('Bob', 'Alex'))
```
