# Invalid object found in `__all__`
**ID:** `PYL-E0604` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PYL-E0604)

![Critical](https://img.shields.io/badge/severity-critical-red) ![Bug Risk](https://img.shields.io/badge/type-bug_risk-green)

The `__all__` list must contain only strings. Having other kinds of objects is an error.


## Bad practice
Trying to do this will give an error:


```python
__all__ = ['calculate', 'patch', diff]  # `diff` is not a string!
```

## Recommended

```python
__all__ = ['calculate', 'patch', 'diff']
```
Note: During Autofix, DeepSource will convert the invalid objects into a string.
