# Dictionary key is unhashable
**ID:** `PYL-E1140` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PYL-E1140)

![Critical](https://img.shields.io/badge/severity-critical-red) ![Bug Risk](https://img.shields.io/badge/type-bug_risk-green)

It is recommended to use immutable objects as keys in a dictionary. If you want to use complex objects, ensure that they are hashable.
