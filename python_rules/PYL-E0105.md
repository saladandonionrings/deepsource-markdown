# `yield` used outside of function
**ID:** `PYL-E0105` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PYL-E0105)

![Critical](https://img.shields.io/badge/severity-critical-red) ![Bug Risk](https://img.shields.io/badge/type-bug_risk-green)

Using a `yield` statement only makes sense in the context of functions. Using it outside a function would raise a `SyntaxError` .


## Bad practice
`yield` in a loop would raise a `SyntaxError` :


```python
for x in range(100):
    yield x
```

## Recommended
But, if this loop is inside a function, it works as it would convert the function into a generator:


```python
def my_func():
    for x in range(100):
        yield x
```
