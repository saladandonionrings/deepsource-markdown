# Audit required: Use of `exec`
**ID:** `PYL-W0122` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PYL-W0122)

![Major](https://img.shields.io/badge/severity-major-orange) ![Security](https://img.shields.io/badge/type-security-red)

Usage of `exec` function is strongly discouraged, since it opens up possibilities of unauthorized code execution if the statements are not escaped properly. Read more on why should `exec` be avoided [here](https://stackoverflow.com/questions/1933451/why-should-exec-and-eval-be-avoided) .
