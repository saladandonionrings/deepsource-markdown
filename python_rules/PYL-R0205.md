# Useless inheritance from `object`
**ID:** `PYL-R0205` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PYL-R0205)

![Major](https://img.shields.io/badge/severity-major-orange) ![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

The class is inheriting from `object` , which is implicit under Python 3 , hence can be safely removed from bases.

Not preferred:


```python
class Foo(object):
    """Foo bar baz"""
```
Preferred:


```python
class Foo:
    """Foo bar baz"""
```
