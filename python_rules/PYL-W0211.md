# Bad `staticmethod` argument detected
**ID:** `PYL-W0211` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PYL-W0211)

![Major](https://img.shields.io/badge/severity-major-orange) ![Bug Risk](https://img.shields.io/badge/type-bug_risk-green)

It is customary for instance or class methods to take `self` or `cls` , respectively, as their first arguments, a method that uses either of these names but is found to be a static method may have been defined incorrectly. Choose names other than `self` or `cls` for arguments to avoid confusing other programmers looking at your code.


## Bad practice

```python
class API:
    @staticmethod
    def get_endpoint(self):  # The `self` here can be confusing
        return self + '/api/v2/users'
```

## Recommended

```python
class API:
    @staticmethod
    def get_endpoint(origin):  # Any other, more suitable variable name
        return origin + '/api/v2/users'
```
