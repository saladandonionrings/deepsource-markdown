# Bad comparison test
**ID:** `PTC-W0023` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PTC-W0023)

![Minor](https://img.shields.io/badge/severity-minor-yellow) ![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

It is recommended to use `identity test` ( `is` ) instead of equality test ( `==` ) when you need to compare types of two objects.


## Bad practice

```python
if type(x) == str:
    print(x)
```

## Recommended

```python
if type(x) is str:
    print(x)
# Alternatively,
if isinstance(x, str):
    print(x)
```
