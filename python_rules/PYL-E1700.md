# `yield from` inside an `async` function
**ID:** `PYL-E1700` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PYL-E1700)

![Major](https://img.shields.io/badge/severity-major-orange) ![Bug Risk](https://img.shields.io/badge/type-bug_risk-green)

Use of `yield from` is not permitted inside an async function. This is a syntax error. It is recommend to rewrite coroutine without using `yield from` .


```python
async def my_coroutine():
    yield 1
    yield from [2, 3, 4, 5, 6] # Syntax Error
```

## Recommended

```python
async def my_coroutine():
    yield 1
    for i in [2, 3, 4, 5, 6]:
        yield i
```
