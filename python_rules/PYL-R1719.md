# `if` expression used can be simplified
**ID:** `PYL-R1719` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PYL-R1719)

![Minor](https://img.shields.io/badge/severity-minor-yellow) ![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

The `if` expression used here can be simplified to increase the code readability. Example:


```python
value = True if arg else False
```
This can be simplified into:


```python
value = bool(arg)
```
