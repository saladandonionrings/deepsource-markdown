# Redundant `default=None` for a model field
**ID:** `PTC-W0907` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PTC-W0907)

![Minor](https://img.shields.io/badge/severity-minor-yellow) ![Style](https://img.shields.io/badge/type-style-blue)

If field is nullable ( `null=True` ), then it is unnecessary to provide `default=None` since this is implicit. It is recommended to remove `default=None` kwarg.


## Not Preferred:

```python
class SomeModel(models.Model):
    some_field = models.EmailField(null=True, default=None)
```

## Preferred:

```python
class SomeModel(models.Model):
    some_field = EmailField(null=True)
```
