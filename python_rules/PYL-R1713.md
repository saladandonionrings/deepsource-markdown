# Consider using `join`
**ID:** `PYL-R1713` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PYL-R1713)

![Major](https://img.shields.io/badge/severity-major-orange) ![Performance](https://img.shields.io/badge/type-performance-white)

Consider using `str.join(sequence)` instead of joining the elements of a sequence using `for` loop iteration. `str.join(sequence)` is faster, uses less memory and increases readability compared to a `for` loop iteration.


## Bad practice

```python
my_string = ""
for element in data:
    my_string += element
```

## Recommended

```python
my_string = "".join(data)
```
