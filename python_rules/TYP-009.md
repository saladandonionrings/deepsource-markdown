# Invalid return `type hint` for generator function
**ID:** `TYP-009` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/TYP-009)

![Major](https://img.shields.io/badge/severity-major-orange) ![Type Check](https://img.shields.io/badge/type-type_check-lightgrey)

The return type of a generator function should be `Generator` or one of its supertypes.
