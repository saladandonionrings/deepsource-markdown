# Invalid `envvar` default
**ID:** `PYL-W1508` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PYL-W1508)

![Major](https://img.shields.io/badge/severity-major-orange) ![Bug Risk](https://img.shields.io/badge/type-bug_risk-green)

Env manipulation functions return `None` or `str` values. Supplying anything different as a default may cause bugs. See [https://docs.python.org/3/library/os.html#os.getenv](https://docs.python.org/3/library/os.html#os.getenv) .


## Bad practice

```python
import os

indent = os.getenv("INDENT_LEVEL", default=4)  # 4 is an int, but getenv gives strings
```

## Recommended

```python
import os

indent_str = os.getenv("INDENT_LEVEL", default="4")  # Now it's a string
indent = int(indent_str)
```
