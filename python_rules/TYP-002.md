# No return value expected
**ID:** `TYP-002` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/TYP-002)

![Major](https://img.shields.io/badge/severity-major-orange) ![Type Check](https://img.shields.io/badge/type-type_check-lightgrey)

Callable is returning a value where no return value was expected.

This usually means you didn't update your function's signature after updating its body, or you accidentally returned something you didn't mean to.


## Bad practice

```python
def main() -> None:
    try:
        run_app()
    except:
        return 1

    return 0
```

## Recommended
Two ways to fix this would be to either update the signature:


```python
def main() -> int:  # Returning an int
    try:
        run_app()
    except:
        return 1

    return 0
```
or, to not return anything instead:


```python
import sys

def main() -> None:
    try:
        run_app()
    except:
        sys.exit(1)  # Instead of returning, we can exit
```
