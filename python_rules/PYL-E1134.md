# Value is not a mapping
**ID:** `PYL-E1134` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PYL-E1134)

![Major](https://img.shields.io/badge/severity-major-orange) ![Bug Risk](https://img.shields.io/badge/type-bug_risk-green)

A non-mapping value is used in place where a mapping is expected. This is an error. It is recommended to review the value and make sure you're not doing something wrong.


## Bad practice

```python
def test(**kwargs):
    print(kwargs)

dict_value = dict(a=1, b=2, c=3)
list_value = [1, 2, 3]
# This is an error, as a mapping was expected.
test(**list_value)
```

## Recommended

```python
def test(**kwargs):
    print(kwargs)

dict_value = dict(a=1, b=2, c=3)
# This is fine
test(**dict_value)
```
