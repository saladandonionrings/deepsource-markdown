# Continuation line with same indent as next logical line
**ID:** `FLK-E125` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/FLK-E125)

![Minor](https://img.shields.io/badge/severity-minor-yellow) ![Style](https://img.shields.io/badge/type-style-blue)

Continuation lines should not be indented at the same level as the next logical line. Instead, they should be indented to one more level so as to distinguish them from the next line.
