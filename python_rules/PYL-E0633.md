# Attempting to unpack a non-sequence object
**ID:** `PYL-E0633` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PYL-E0633)

![Critical](https://img.shields.io/badge/severity-critical-red) ![Bug Risk](https://img.shields.io/badge/type-bug_risk-green)

Unpacking is supported only for sequence objects, like lists, sets, tuples, iterators or generators, etc.


## Bad practice

```python
def get_system_info():
    return ['linux', '64-bit']

system, architecture = get_system_info  # is a function, cannot be unpacked
print(system)
```

## Recommended

```python
def get_system_info():
    return ['linux', '64-bit']

system, architecture = get_system_info()  # returns a list, can be unpacked
print(system)
```
