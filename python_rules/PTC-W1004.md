# Audit required: Insecure cipher
**ID:** `PTC-W1004` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PTC-W1004)

![Major](https://img.shields.io/badge/severity-major-orange) ![Security](https://img.shields.io/badge/type-security-red)

Cipher used is not secure. It is recommended to replace it with a known secure cipher such as `AES` .

Following ciphers are considered weak for a variety of reasons:

* [Single DES](https://pycryptodome.readthedocs.io/en/latest/src/cipher/des.html) : DES was never cryptographically broken, but its key length is too short by nowadays standards and it could be brute forced with some effort.
* [BlowFish](https://cryptography.io/en/latest/hazmat/primitives/symmetric-encryption/#cryptography.hazmat.primitives.ciphers.algorithms.Blowfish) : Susceptible to attacks when using weak keys.
* [ARC4](https://cryptography.io/en/latest/hazmat/primitives/symmetric-encryption/#cryptography.hazmat.primitives.ciphers.algorithms.ARC4) : Weaknesses in its initial stream output.
* [IDEA](https://cryptography.io/en/latest/hazmat/primitives/symmetric-encryption/#cryptography.hazmat.primitives.ciphers.algorithms.IDEA) : Susceptible to attacks when using weak keys.

New applications should avoid their use and existing applications should strongly consider migrating away.

It is recommended to use [AES](https://cryptography.io/en/latest/hazmat/primitives/symmetric-encryption/#cryptography.hazmat.primitives.ciphers.algorithms.AES) . AES is both fast and cryptographically strong. It is a good default choice for encryption.


## Bad practice
[pycryptodome library](https://pycryptodome.readthedocs.io/en/latest/index.html) :


```python
from Crypto.Cipher import ARC4, DES

cipher1 = ARC4.new(key)
cipher2 = DES.new(key, DES.MODE_OFB)
```
[pyca/cryptography library](https://cryptography.io/en/latest/) :


```python
from cryptography.hazmat.primitives.ciphers import Cipher, algorithms, modes
cipher = Cipher(algorithms.Blowfish(key), modes.CBC(iv))
```

## Recommended
pycryptodome library:


```python
from Crypto.Cipher import AES

cipher1 = AES.new(key, AES.MODE_EAX)
cipher2 = AES.new(key, AES.MODE_OFB)
```
pyca/cryptography library:


```python
from cryptography.hazmat.primitives.ciphers import Cipher, algorithms, modes

cipher = Cipher(algorithms.AES(key), modes.CBC(iv))
```
