# `break` statement used outside of a `while` or `for` loop
**ID:** `FLK-F701` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/FLK-F701)

![Critical](https://img.shields.io/badge/severity-critical-red) ![Bug Risk](https://img.shields.io/badge/type-bug_risk-green)

The `break` statement cannot not be used outside a loop. This will raise a `SyntaxError`
