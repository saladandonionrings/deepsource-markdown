# Exception context set to something which is not an exception, nor None
**ID:** `PYL-E0703` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PYL-E0703)

![Critical](https://img.shields.io/badge/severity-critical-red) ![Bug Risk](https://img.shields.io/badge/type-bug_risk-green)

The syntax `raise ... from ...` can be used only when the exception context is an exception, or `None` . Trying to do so with other object types is an error.
