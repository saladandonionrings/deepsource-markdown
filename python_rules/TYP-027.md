# All type variables should be listed here
**ID:** `TYP-027` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/TYP-027)

![Major](https://img.shields.io/badge/severity-major-orange) ![Type Check](https://img.shields.io/badge/type-type_check-lightgrey)

If you include `Generic[...]` or `Protocol[...]` in bases, then it should list all type variables present in other bases (or more, if needed). Read more about this [here](https://mypy.readthedocs.io/en/latest/generics.html#defining-sub-classes-of-generic-classes) .
