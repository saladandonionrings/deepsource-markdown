# Misplaced comparison constant
**ID:** `PYL-C2201` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PYL-C2201)

![Major](https://img.shields.io/badge/severity-major-orange) ![Style](https://img.shields.io/badge/type-style-blue)

The constant is placed on the left side of a comparison. It is usually clearer in intent to place it in the right hand side of the comparison.


## Bad practice

```python
if 0 == x % 2:
    ...
```

## Recommended:

```python
if x % 2 == 0:
    ...
```
