# Expected 2 blank lines after end of function or class
**ID:** `FLK-E305` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/FLK-E305)

![Minor](https://img.shields.io/badge/severity-minor-yellow) ![Style](https://img.shields.io/badge/type-style-blue)

Functions and classes should have two blank lines after them, separating them from other functions and classes.
