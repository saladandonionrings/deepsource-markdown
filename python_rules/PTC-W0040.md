# Using deprecated method `assertEquals`
**ID:** `PTC-W0040` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PTC-W0040)

![Major](https://img.shields.io/badge/severity-major-orange) ![Bug Risk](https://img.shields.io/badge/type-bug_risk-green)

`assertEquals` has been deprecated. Use the standardized `assertEqual` instead.


## Bad practice

```python
import unittest

class Tests(unittest.TestCase):
    def test_something(self, arg1, arg2):
        self.assertEquals(arg1, arg2)
```

## Recommended

```python
import unittest

class Tests(unittest.TestCase):
    def test_something(self, arg1, arg2):
        self.assertEqual(arg1, arg2)
```
