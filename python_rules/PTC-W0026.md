# Found `yield` statement inside a generator expression
**ID:** `PTC-W0026` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PTC-W0026)

![Major](https://img.shields.io/badge/severity-major-orange) ![Bug Risk](https://img.shields.io/badge/type-bug_risk-green)

It is redundant to use `yield` inside a generator expression as it will return a generator object just like a generator expression. **Note** : Use of `yield` inside a generator expression was allowed until Python 3.7 but is a syntax error from Python 3.8 onwards. It is recommended to remove the yield statement here.


## Bad practice

```python
number_generator = ((yield num) for num in range(1,1000))
```

## Recommended

```python
number_generator = (num for num in range(1,1000))
```
