# Class has duplicate bases
**ID:** `PYL-E0241` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PYL-E0241)

![Critical](https://img.shields.io/badge/severity-critical-red) ![Bug Risk](https://img.shields.io/badge/type-bug_risk-green)

A class must not specify inheritance from another class more than once. Duplicate base classes will raise a `TypeError` .


## Bad practice

```python
class SomeClass(SomeBase, Unique, SomeBase):
    ...
```

## Recommended

```python
class SomeClass(SomeBase, Unique):
    ...
```
