# Invalid slots object
**ID:** `PYL-E0238` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PYL-E0238)

![Critical](https://img.shields.io/badge/severity-critical-red) ![Bug Risk](https://img.shields.io/badge/type-bug_risk-green)

An invalid slots value is found in class. Only a string, an iterable or a sequence is permitted. This is an error.


```python
class Bad:
    __slots__ = list
```
Assigning an invalid object to `__slots__` would raise a `TypeError` . During autofix, DeepSource will replace the invalid object with `[]` .
