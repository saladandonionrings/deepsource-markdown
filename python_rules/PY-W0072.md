# Appending to dictionary immediately following its definition
**ID:** `PY-W0072` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PY-W0072)

![Major](https://img.shields.io/badge/severity-major-orange) ![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

Instead of adding items to a dictionary just after creation, it should be refactored to add those items into the dict definition itself.


## Bad practice

```python
env = {}
env['HOST'] = 'example.com'
env['PORT'] = 8080
```

## Recommended

```python
env = {'HOST': 'example.com', 'PORT': 8080}
```
