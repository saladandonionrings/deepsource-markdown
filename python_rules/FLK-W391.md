# Multiple blank lines detected at end of the file
**ID:** `FLK-W391` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/FLK-W391)

![Minor](https://img.shields.io/badge/severity-minor-yellow) ![Style](https://img.shields.io/badge/type-style-blue)

There should be one, and only one, blank line at the end of each file. This warning will occur when there are two or more blank lines.
