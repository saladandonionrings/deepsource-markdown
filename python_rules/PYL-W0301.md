# Unnecessary semicolon
**ID:** `PYL-W0301` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PYL-W0301)

![Minor](https://img.shields.io/badge/severity-minor-yellow) ![Style](https://img.shields.io/badge/type-style-blue)

Semicolons are not needed in Python unless using multiple statements in single line. Use of semi-colons add unnecessary clutter.


## Bad practice

```python
for x in range(10):
    if input[x]:
        i = x - 2;
        x = i * i;
    print(i);
```

## Preferred

```python
for x in range(10):
    if input[x]:
        i = x - 2
        x = i * i
    print(i)
```
