# Non ascii bytes literals detected
**ID:** `PYL-E1610` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PYL-E1610)

![Major](https://img.shields.io/badge/severity-major-orange) ![Bug Risk](https://img.shields.io/badge/type-bug_risk-green)

Non-ascii byte literals are being used in this file. They are no longer supported in Python 3.


## Bad practice

```python
some_var = b"國華"
```

## Recommended

```python
some_var = b"Guohua"
```
