# Invalid `Literal[...]` type hint
**ID:** `TYP-041` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/TYP-041)

![Major](https://img.shields.io/badge/severity-major-orange) ![Type Check](https://img.shields.io/badge/type-type_check-lightgrey)

Arbitrary arguments are given to `Literal[...]` , which makes it an invalid type. It is recommended to remove the arbitrary arguments.
