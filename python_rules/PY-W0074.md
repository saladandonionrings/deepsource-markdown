# Consider using `any`
**ID:** `PY-W0074` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PY-W0074)

![Major](https://img.shields.io/badge/severity-major-orange) ![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

It is recommended to use `any` for this loop, as it will make your code smaller and more readable.


## Bad practice

```python
def has_ten(my_list):
    for item in my_list:
        if item == 10:
            return True
    return False
```

## Recommended

```python
def has_ten(my_list):
    return any(item == 10 for item in my_list)
```
