# Used a variable that was bound inside a comprehension
**ID:** `PYL-W1662` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PYL-W1662)

![Minor](https://img.shields.io/badge/severity-minor-yellow) ![Bug Risk](https://img.shields.io/badge/type-bug_risk-green)

Accessed a variable, that was bound in a comprehension handler, outside of the comprehension itself. On Python 3 these variables will be deleted outside of the comprehension, and will give an error on being used.
