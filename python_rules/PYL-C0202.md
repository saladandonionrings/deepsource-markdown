# Bad classmethod argument
**ID:** `PYL-C0202` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PYL-C0202)

![Major](https://img.shields.io/badge/severity-major-orange) ![Style](https://img.shields.io/badge/type-style-blue)

It is recommended to name the first argument of a classmethod as `cls` so that it's easy to differentiate them from regular instance methods.


## Bad practice

```python
class SpaceShip:

    @classmethod
    def trajectory(self, x, y, z):  # `self` here can be very confusing
        ...
```

## Recommended

```python
class SpaceShip:

    @classmethod
    def trajectory(cls, x, y, z):
        ...
```
