# Old `raise` syntax deteted
**ID:** `PYL-E1604` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PYL-E1604)

![Major](https://img.shields.io/badge/severity-major-orange) ![Bug Risk](https://img.shields.io/badge/type-bug_risk-green)

The alternate raise syntax `raise SomeError, context` is used instead of `raise SomeError(context)` . This has been deprecated and is a syntax error in Python 3.


## Bad practice

```python
raise SomeError, context
```

## Recommended

```python
raise SomeError(context)
```
