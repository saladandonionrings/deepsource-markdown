# Unnecessary parentheses after keyword
**ID:** `PYL-C0325` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PYL-C0325)

![Minor](https://img.shields.io/badge/severity-minor-yellow) ![Style](https://img.shields.io/badge/type-style-blue)

Extra parentheses in code can be removed for improved readability.

In the examples below, the first example is more readable than the second one.

Preferred:


```python
if len(some_variable) + 1 < 1:
    ...
```
Not preferred:


```python
if (len(some_variable) + 1) < 1:
    ...
```
