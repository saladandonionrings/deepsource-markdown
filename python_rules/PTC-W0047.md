# Empty block of code found
**ID:** `PTC-W0047` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PTC-W0047)

![Major](https://img.shields.io/badge/severity-major-orange) ![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

In most cases, an empty body of `for` , `while` or `if` implies some piece of code is missing. Such empty block must be either filled or removed.


## Bad practice

```python
for i in range(10):
    pass
```

## Preferred:

```python
for i in range(10):
    print(i)
```
