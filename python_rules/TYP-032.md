# Unsupported `Type` declaration
**ID:** `TYP-032` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/TYP-032)

![Major](https://img.shields.io/badge/severity-major-orange) ![Type Check](https://img.shields.io/badge/type-type_check-lightgrey)

Declaring `Type` during an assignment to a non-self attribute is not supported.
