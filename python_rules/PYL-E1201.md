# Logging format string terminated in between conversion specifier
**ID:** `PYL-E1201` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PYL-E1201)

![Critical](https://img.shields.io/badge/severity-critical-red) ![Bug Risk](https://img.shields.io/badge/type-bug_risk-green)

Logging statement format string shall not terminate before the end of a conversion specifier.
