# Use `tuple` unpacking to swap variables
**ID:** `PYL-R1712` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PYL-R1712)

![Major](https://img.shields.io/badge/severity-major-orange) ![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

It is recommended not to use a temporary variable in order to swap variables. Using tuple unpacking to directly swap variables makes the intention more clear.


## Bad practice
Not preferred:


```python
temp = a
a = b
b = temp
```

## Recommended

```python
a, b = b, a
```
swaps the values of `a` and `b` .
