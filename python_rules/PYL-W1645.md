# `message` attribute accessed on Exception
**ID:** `PYL-W1645` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PYL-W1645)

![Major](https://img.shields.io/badge/severity-major-orange) ![Bug Risk](https://img.shields.io/badge/type-bug_risk-green)

`Exception.message` has been removed in Python 3. It is recommended to use `str(exception)` instead to access the exception message.
