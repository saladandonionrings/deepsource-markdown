# Imports from same package are not grouped
**ID:** `PYL-C0412` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PYL-C0412)

![Minor](https://img.shields.io/badge/severity-minor-yellow) ![Style](https://img.shields.io/badge/type-style-blue)

Imports for the same package are scattered and not grouped together. It is recommended to keep the imports from the same package together. It makes the code easier to read.

Not preferred:


```python
import os
import sys
from os import listdir
import time
from os.path import isfile, join
```
Preferred:


```python
import os
from os import listdir
from os.path import isfile, join

import sys
import time
```
