# Variable assigned to itself
**ID:** `PYL-W0127` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PYL-W0127)

![Minor](https://img.shields.io/badge/severity-minor-yellow) ![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

Variable is being assigned to itself.

It is unnecessary to assign a variable to itself without any modification. You can safely remove this assignment operation.


## Bad practice:

```python
if condition:
    x = x
    doSomething()
```

## Recommended

```python
if condition:
    doSomething()
```
