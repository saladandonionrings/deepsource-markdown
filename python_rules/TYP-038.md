# Deprecated type hint used
**ID:** `TYP-038` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/TYP-038)

![Major](https://img.shields.io/badge/severity-major-orange) ![Type Check](https://img.shields.io/badge/type-type_check-lightgrey)

The special form `Any(...)` has been deprecated. Use `cast(Any, ...)` instead.
