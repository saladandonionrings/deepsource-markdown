# Inline comment should start with `# `
**ID:** `FLK-E262` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/FLK-E262)

![Minor](https://img.shields.io/badge/severity-minor-yellow) ![Style](https://img.shields.io/badge/type-style-blue)

Inline comments should have one space before the pound sign `#` and the comment itself.
