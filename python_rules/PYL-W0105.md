# Unassigned string statement
**ID:** `PYL-W0105` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PYL-W0105)

![Minor](https://img.shields.io/badge/severity-minor-yellow) ![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

The string statement has not been assigned to anything. This is pointless and should be removed if not necessary. In case this is supposed to describe what's happening in the code, it is recommended to use comments or docstrings instead.


```python
'Define a function to add 2 numbers here.'
def add(a, b):
    return a + b
```
Use Comments:


```python
# Define a function to add 2 numbers here.
def add(a, b):
    return a + b
```
or, use docstrings:


```python
def add(a, b):
    '''Function to add 2 numbers here.'''
    return a + b
```
