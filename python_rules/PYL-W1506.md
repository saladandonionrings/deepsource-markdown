# Thread has no target function
**ID:** `PYL-W1506` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PYL-W1506)

![Major](https://img.shields.io/badge/severity-major-orange) ![Bug Risk](https://img.shields.io/badge/type-bug_risk-green)

`threading.Thread` class is instantiated without the target function being passed. By default, the first parameter is the group param, not the target param. You can read more about this [here](https://docs.python.org/3/library/threading.html#threading.Thread) .


## Bad practice

```python
import threading

def do_work():
    for _ in range(10000):
        pass

my_thread = threading.Thread(do_work)
```

## Recommended

```python
import threading

def do_work():
    for _ in range(10000):
        pass

my_thread = threading.Thread(target=do_work)
```
