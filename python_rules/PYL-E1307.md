# Bad string format type
**ID:** `PYL-E1307` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PYL-E1307)

![Major](https://img.shields.io/badge/severity-major-orange) ![Bug Risk](https://img.shields.io/badge/type-bug_risk-green)

The type required by this format string is not the same as the actual argument type. This can cause an error.


## Bad practice

```python
print("%d" % "Cat")  # %d expects an integer.
```

## Recommended

```python
print("%s" % "Cat")
print("%d" % 43)
```
