# Function contains unused argument
**ID:** `PYL-W0613` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PYL-W0613)

![Major](https://img.shields.io/badge/severity-major-orange) ![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

An unused argument can lead to confusions. It should be removed. If this variable is necessary, name the variable `_` or start the name with `unused` or `_unused` .


## Bad practice

```python
def square(x, y=1):
    return x * x

class MySubClass(MyClass):
    def __init__(self, number):
        self.value = 42  # argument `number` remains unused
```

## Preferred:

```python
def square(x):
    return x * x

class MySubClass(MyClass):
    def __init__(self, _):
        self.value = 42
```
