# Bad metaclass-method argument
**ID:** `PYL-C0203` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PYL-C0203)

![Major](https://img.shields.io/badge/severity-major-orange) ![Style](https://img.shields.io/badge/type-style-blue)

Use of `cls` as the first named argument in a metaclass method is recommended so it's easy to differentiate them from regular instance methods.


## Not preferred

```python
class MyMeta(type):
    def some_method(self):
       ...
```

## Preferred

```python
class MyMeta(type):
    def some_methon(cls):
        ...
```
