# Missing return value
**ID:** `TYP-006` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/TYP-006)

![Major](https://img.shields.io/badge/severity-major-orange) ![Type Check](https://img.shields.io/badge/type-type_check-lightgrey)

This callable is expected to return a value but nothing is being returned. Either make sure you return a value, or change the annotation to `-> None` .


## Bad practice

```python
def magic_number() -> int:
    42  # Missing `return`
```

## Recommended

```python
def magic_number() -> int:
    return 42
```
