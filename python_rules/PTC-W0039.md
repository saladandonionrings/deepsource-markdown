# Unnecessary `None` provided as default
**ID:** `PTC-W0039` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PTC-W0039)

![Minor](https://img.shields.io/badge/severity-minor-yellow) ![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

It is unnecessary to provide `None` as the default value when the key is not present in the dictionary as `get` implicitly returns `None` .

Not preferred:


```python
value = some_dict.get("key", None)
```
Preferred:


```python
value = some_dict.get("key")
```
