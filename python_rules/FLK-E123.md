# Closing bracket does not match indentation of opening bracket's line
**ID:** `FLK-E123` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/FLK-E123)

![Minor](https://img.shields.io/badge/severity-minor-yellow) ![Style](https://img.shields.io/badge/type-style-blue)

Closing brackets should match the same indentation level of the line that their opening bracket started on.
