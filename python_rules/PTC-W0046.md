# Unit test class with no tests
**ID:** `PTC-W0046` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PTC-W0046)

![Major](https://img.shields.io/badge/severity-major-orange) ![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

Test methods should always start with the `test` . If there are no such methods, the class overriding `unittest.TestCase` won't run any test.

Not preferred:


```python
import unittest

def Tests(unittest.TestCase):
    def my_test(self, arg1, arg2):
        self.assertEquals(arg1, arg2)
```
Preferred:


```python
import unittest

def Tests(unittest.TestCase):
    def test_something(self, arg1, arg2):
        self.assertEquals(arg1, arg2)
```
