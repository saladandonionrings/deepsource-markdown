# Argument passed both by position and keyword in function call
**ID:** `PYL-E1124` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PYL-E1124)

![Critical](https://img.shields.io/badge/severity-critical-red) ![Bug Risk](https://img.shields.io/badge/type-bug_risk-green)

The function call would result in assigning multiple values to a function parameter, one value from a positional argument and one from a keyword argument.


## Bad practice

```python
def square(a):
    return a*a

res = square(2, a=3)
```

## Recommended:

```python
def square(a):
    return a*a

res = square(a=3)
```
