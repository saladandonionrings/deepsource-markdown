# Invalid `slots` object
**ID:** `PYL-E0236` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PYL-E0236)

![Critical](https://img.shields.io/badge/severity-critical-red) ![Bug Risk](https://img.shields.io/badge/type-bug_risk-green)

`__slots__` must contain only non-empty strings or an iterable/sequence of strings with variable names used by instances, otherwise it will raise a `TypeError` during runtime. It is recommended to review the `__slots__` objects and make sure they are all strings.


## Problematic code:

```python
class ExampleClass:
    __slots__ = ("a", "b", "", 34)
```

## Recommended

```python
class ExampleClass:
    __slots__ = ("a", "b")
```
