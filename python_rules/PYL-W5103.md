# Model does not explicitly define `__unicode__`
**ID:** `PYL-W5103` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PYL-W5103)

![Minor](https://img.shields.io/badge/severity-minor-yellow) ![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

Django models should implement a `__unicode__` method for string representation. A parent class of this model does, but ideally all models should be explicit.
