# Adding to set immediately following its definition
**ID:** `PY-W0071` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PY-W0071)

![Major](https://img.shields.io/badge/severity-major-orange) ![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

Instead of addepts items to a set just after creation, it should be refactored to add those items into the set definition itself.


## Bad practice

```python
trees = set()
trees.add('cedar')
trees.add('oak')
```

## Recommended

```python
trees = {'cedar', 'oak'}
```
