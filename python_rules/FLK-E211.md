# Whitespace before opening parenthesis
**ID:** `FLK-E211` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/FLK-E211)

![Minor](https://img.shields.io/badge/severity-minor-yellow) ![Style](https://img.shields.io/badge/type-style-blue)

Open parentheses should not have any space before or after them.
