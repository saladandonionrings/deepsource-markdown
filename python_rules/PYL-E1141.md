# Missing `.items()`
**ID:** `PYL-E1141` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PYL-E1141)

![Major](https://img.shields.io/badge/severity-major-orange) ![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

You're trying to iterate through a dictionary without using the `.items()` (for Python 3) or `.iteritems()` (Python 2) method. This is a `ValueError` .


## Bad practice

```python
data = {'x': 1, 'y': 2, 'z': 3}
for key, value in data:
    print(f'key: {key} => value: {value}')
```

## Recommended

```python
data = {'x': 1, 'y': 2, 'z': 3}
for key, value in data.items():
    print(f'key: {key} => value: {value}')
```
