# Invalid sequence index found
**ID:** `PYL-E1126` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PYL-E1126)

![Critical](https://img.shields.io/badge/severity-critical-red) ![Bug Risk](https://img.shields.io/badge/type-bug_risk-green)

A sequence type is indexed with an invalid type. Valid types are `int` s, slices, and objects with an `__index__` method.
