# `if` statement can be simplified
**ID:** `PYL-R1703` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PYL-R1703)

![Major](https://img.shields.io/badge/severity-major-orange) ![Anti-pattern](https://img.shields.io/badge/type-anti_pattern-purple)

The highlighted `if` block can be refactored into a simplified statement, increasing its readability.


## Bad practice

```python
def simplifiable_if(arg, arg2):
    if arg and not arg2:
        var = True
    else:
        var = False

    return var
```

## Recommended

```python
def simplifiable_if(arg, arg2):
    var = bool(arg and not arg2)

    return var
```
