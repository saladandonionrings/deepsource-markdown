# Use of a non-existent operator detected
**ID:** `PYL-E0107` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PYL-E0107)

![Major](https://img.shields.io/badge/severity-major-orange) ![Bug Risk](https://img.shields.io/badge/type-bug_risk-green)

Detected the use of one of the C-style pre/post increment or decrement operators, `--` and `++` , which don't exist in Python. Use the `+=` or `-=` operators instead.

This will cause an error:


```python
x = 50
x++
```
But, doing this is okay:


```python
x = 50
x += 1
```
