# Use of `cmp` argument detected
**ID:** `PYL-W1640` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PYL-W1640)

![Major](https://img.shields.io/badge/severity-major-orange) ![Bug Risk](https://img.shields.io/badge/type-bug_risk-green)

A custom comparator is being passed for sorting. In Python 3.0, the `cmp` parameter was removed entirely (as part of a larger effort to simplify and unify the language, eliminating the conflict between rich comparisons and the `__cmp__` methods).
