# Object which is not a context manager used with the `with` statement
**ID:** `PYL-E1129` | **Lien:** [DeepSource](https://deepsource.com/directory/python/issues/PYL-E1129)

![Critical](https://img.shields.io/badge/severity-critical-red) ![Bug Risk](https://img.shields.io/badge/type-bug_risk-green)

The object used in the `with` statement doesn't implement the context manager protocol (enter/exit). This is an error.
